# 41. 대화 장면 프롬프트 공식

VEO 3의 가장 강력한 기능 중 하나는 **캐릭터가 실제로 말하는 영상**을 만들 수 있다는 것입니다. 단순한 텍스트 자막이 아니라, 입 모양이 대사와 동기화되고 감정 표현까지 담긴 진짜 대화 장면을 생성할 수 있습니다.

---

## 대화 장면 프롬프트 공식

```
[캐릭터] + [행동] + [카메라 앵글] + [환경/조명] + [대화] + [목소리 톤] + [배경음/음악(선택)]
```

또는 더 자연스러운 문장형:

```
[캐릭터]이/가 [행동]한다. [카메라 앵글] 샷. [환경/조명].
[캐릭터]이/가 [목소리 톤]으로 말한다: [대화 내용]
```

---

## 공식의 각 요소

### 1. 캐릭터 (Character)
누가 말하는지 명확히 설명합니다.

```
예시:
- 짧은 은발에 낡은 코트를 입은 중년 탐정
- 헝클어진 붉은 털을 가진 키 큰 빅풋
- 흰색 갑옷을 착용한 제국 스톰트루퍼
```

### 2. 행동 (Action)
말하는 동안 캐릭터가 무엇을 하는지 지정합니다.

```
예시:
- 카메라를 정면으로 바라보며
- 셀카봉을 들고 걸으면서
- 창문을 내다보며 돌아서서
```

### 3. 카메라 앵글 (Camera Angle)
대화 장면에 가장 잘 맞는 앵글을 선택합니다.

| 앵글 | 효과 |
|------|------|
| 미디엄 샷 | 균형 잡힌 대화 장면 |
| 클로즈업 | 감정 강조 |
| 셀카 앵글 | 브이로그, 친근한 느낌 |
| OTS | 대화 상대방 관점 |

### 4. 환경 / 조명 (Environment / Lighting)
장면의 배경과 빛을 설정합니다.

```
예시:
- 눈보라가 치는 빽빽한 숲속
- 차가운 형광등이 켜진 심문실
- 황혼의 도시 옥상
```

### 5. 대화 내용 (Dialogue)
캐릭터가 말할 정확한 대사를 작성합니다.

> ⚠️ **중요:** 대사에 따옴표("")를 사용하면 자막이 영상 위에 표시될 수 있습니다. 일반 텍스트로 작성하세요.

```
❌ 피하세요: "안녕하세요, 저는 탐정입니다."
✅ 권장: 안녕하세요, 저는 탐정입니다.
```

### 6. 목소리 톤 (Voice Tone)
말하는 방식과 감정을 지정합니다.

```
예시:
- 깊고 울림 있는 스코틀랜드 남성 목소리로
- 긴장해서 약간 떨리는 젊은 여성 목소리로
- 자신감 있고 강렬한 군인 목소리로
- 피곤하고 건조한 중년 남성 목소리로
```

### 7. 배경음 (선택사항)
환경 소음이나 배경 음악을 추가합니다.

```
예시:
- 배경음: 거센 눈보라, 나뭇가지 부러지는 소리
- 배경음: 도시 소음, 멀리서 들리는 사이렌
- 배경 음악: 부드러운 피아노 멜로디
```

---

## 완성 예시 — 형사 심문 장면

```
짧은 은발에 낡은 갈색 코트를 입은 50대 탐정이
나무 의자에 앉아 카메라를 정면으로 바라본다.
미디엄 샷. 낡은 심문실, 차가운 형광등 조명,
테이블 위에 서류 더미.
탐정이 지치고 낮고 건조한 목소리로 말한다:
당신이 그날 밤 어디에 있었는지는 이미 알고 있어요.
그냥 솔직하게 말해주면 됩니다.
배경음: 시계 초침 소리, 멀리서 들리는 복도 발소리.
```

---

## 완성 예시 — 브이로그 캐릭터

```
키가 크고 덥수룩한 짙은 적갈색 털, 영장류 같은 얼굴을 가진 빅풋이
셀카봉을 들고 눈보라 치는 숲속을 걷고 있다.
셀카 카메라 앵글, 흔들리는 핸드헬드 스타일.
빅풋이 깊고 울림 있는 스코틀랜드 남성 목소리로 말한다:
얘들아 오늘 날씨가 장난 아니야. 발가락이 완전히 얼어붙을 것 같아, 솔직히.
배경음: 강한 바람 소리, 나뭇가지 삐걱거림.
```

---

## 핵심 팁 요약

- 대사는 짧을수록 립싱크 정확도가 높습니다 (1~2문장 권장)
- 목소리 톤을 국적, 나이, 감정까지 구체적으로 지정하면 더 자연스럽습니다
- 배경음은 1~2가지만 선택하세요 (너무 많으면 AI가 혼동)
- 대화 장면은 **텍스트→비디오 변환**에서만 작동합니다 (참조 이미지 사용 시 불가)
