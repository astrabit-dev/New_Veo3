# Veo 3.1 & Veo 3.1 Lite 완전 가이드

> **2026년 3월, Google이 Veo 3를 업그레이드했습니다. 무엇이 달라졌고, Lite는 어떤 모델인가?**

---

## Veo 3.1이란?

**Veo 3.1**은 Google DeepMind가 2026년 3월 발표한 Veo 3의 업그레이드 버전입니다. Veo 3의 핵심 강점(오디오 통합, 립싱크, 영화적 카메라 워크)을 그대로 유지하면서 다음 세 가지를 대폭 개선했습니다.

- **프롬프트 이해력 향상**: 복잡하고 긴 프롬프트를 더 정확하게 해석합니다
- **캐릭터 일관성 강화**: 동일 캐릭터가 영상 전반에 걸쳐 더 안정적으로 유지됩니다
- **오디오 품질 개선**: 대화 자연스러움, 환경음 디테일, 음악 조화가 향상됩니다

---

## Veo 3.1 Lite란?

**Veo 3.1 Lite** (일부 인터페이스에서 Veo 3.1 Fast로 표시)는 Veo 3.1의 경량화 버전입니다. Veo 2 기반의 Fast Mode와 완전한 Veo 3.1 사이에 위치하는 **중간 등급 모델**입니다.

### Veo 3.1 Lite의 가장 큰 차이

이전의 Fast Mode(Veo 2)는 **묵음 영상**만 생성했습니다. Veo 3.1 Lite는 **기본 수준의 오디오**를 포함합니다. 즉, 비용을 절약하면서도 환경음과 간단한 배경음악을 영상에 담을 수 있습니다.

---

## 세 모델 완전 비교표

| 항목 | Fast Mode (Veo 2) | Veo 3.1 Lite | Veo 3.1 (고품질) |
|------|------------------|--------------|-----------------|
| **크레딧** | ~10-20 크레딧 | ~30-50 크레딧 | ~80-100 크레딧 |
| **생성 속도** | 30초-1분 | 1-2분 | 2-5분 |
| **오디오** | ❌ 없음 | ✅ 기본 (환경음, 음악) | ✅ 완전 (대화, 환경음, 음악) |
| **대화/립싱크** | ❌ 없음 | ⚠️ 제한적 | ✅ 우수 |
| **영상 품질** | 좋음 | 매우 좋음 | 최상 |
| **캐릭터 일관성** | 기본 | 향상됨 | 최상 |
| **프롬프트 이해** | 기본 | 향상됨 | 최상 |
| **권장 용도** | 테스트용 | SNS·중간 품질 | 최종 납품용 |

## Veo 3.1 Lite가 채우는 공백

기존 시스템에는 다음과 같은 공백이 있었습니다.

```
Fast Mode (묵음, 저화질) ←————— 큰 격차 —————→ Veo 3 (풀 오디오, 최고화질)
     ~15 크레딧                                    ~100 크레딧
```

Veo 3.1 Lite는 이 격차를 채웁니다.

```
Fast Mode → Veo 3.1 Lite → Veo 3.1
  ~15크레딧    ~40크레딧     ~90크레딧
  묵음         기본 오디오    풀 오디오
```

SNS 콘텐츠처럼 "배경음악은 있어야 하지만 최고 품질까지는 필요 없는" 경우에 Veo 3.1 Lite가 이상적입니다.

## Veo 3 → Veo 3.1 주요 개선 사항

### 1. 프롬프트 이해력

**Veo 3:**
```
A woman in a red dress walks through a busy Tokyo market at night,
neon lights reflecting on wet pavement.
→ 전반적으로 정확하나 일부 디테일 누락
```

**Veo 3.1:**
```
같은 프롬프트
→ 드레스 색상, 시장 분위기, 네온 반사, 보행 속도 등 디테일이 더 정밀하게 반영됨
```

### 2. 캐릭터 일관성

Veo 3에서 같은 캐릭터를 다른 클립에서 생성하면 외모가 달라지는 문제가 있었습니다. Veo 3.1에서는 동일 시리즈 내 캐릭터 외형 유지율이 향상되었습니다.

### 3. 오디오 자연스러움

Veo 3.1에서 대화 음성의 억양과 감정 표현이 더 자연스러워졌으며, 환경음과 음악 간의 볼륨 밸런스가 개선되었습니다.

## 언제 어떤 모델을 쓸까?

### Fast Mode (Veo 2) — 아직 유용한 상황
- 새 프롬프트 아이디어를 **빠르게 테스트**할 때
- 오디오가 **완전히 불필요한** 순수 배경 영상
- 크레딧이 거의 없어 **비용 최소화**가 필요할 때

### Veo 3.1 Lite — 이런 상황에 최적
- TikTok, Instagram Reels, YouTube Shorts용 **SNS 콘텐츠**
- 배경음악이나 환경음은 있어야 하지만 대화는 없는 영상
- 아이디어 테스트 수준을 넘어 **준완성 품질**이 필요할 때
- 한 달에 **여러 편**을 제작해야 해서 크레딧이 중요할 때

### Veo 3.1 고품질 — 반드시 사용해야 할 상황
- 유튜브 업로드, 클라이언트 납품 등 **최종 공개용** 영상
- 캐릭터가 **직접 대사를 말하는** 장면
- 브랜드 광고처럼 **영상 품질이 신뢰에 영향**을 주는 경우
- 포트폴리오에 넣을 **대표작**

## 업데이트된 모델 선택 플로차트

```
새 영상 작업 시작
        ↓
   [처음 시도하는 프롬프트?]
   YES → Fast Mode로 테스트
   NO  → 아래로 이동
        ↓
   [Fast Mode 결과 방향성이 맞는가?]
   NO  → 프롬프트 수정 후 다시 Fast Mode
   YES → 아래로 이동
        ↓
   [최종 공개/납품용 영상인가?]
   NO  → 아래로 이동
   YES → 아래로 이동
        ↓
   [캐릭터 대사/립싱크가 필요한가?]
   YES → Veo 3.1 고품질
   NO  →    ↓
        [오디오(환경음·음악)는 필요한가?]
        YES → Veo 3.1 Lite
        NO  → Fast Mode (비공개 내부용이면 충분)
```

## 크레딧 효율 비교

월 1,000 크레딧 기준으로 생성 가능한 클립 수:

| 전략 | 방법 | 클립 수 |
|------|------|--------|
| Veo 3.1 고품질만 | 1,000 ÷ 90 | 약 11개 |
| Veo 3.1 Lite만 | 1,000 ÷ 40 | 약 25개 |
| Fast Mode만 | 1,000 ÷ 15 | 약 67개 |
| **혼합 전략 (권장)** | Fast 400 + Lite 400 + 고품질 200 | **Fast 27 + Lite 10 + 고품질 2개** |

혼합 전략: Fast Mode로 테스트 → Veo 3.1 Lite로 중간 검증 → Veo 3.1로 최종 완성

---

## Google Flow에서 모델 선택하는 방법

Google Flow의 영상 생성 패널에서 모델 드롭다운을 열면 다음 옵션이 표시됩니다:

- **Veo 3.1** → 고품질 모드 (약 90 크레딧)
- **Veo 3.1 Fast** 또는 **Veo 3.1 Lite** → 경량 모드 (약 40 크레딧)
- **Fast** 또는 **Veo 2** → 기존 Fast Mode (약 15 크레딧)

> **인터페이스 표기 주의:** Google Flow UI 업데이트에 따라 "Veo 3.1 Fast"와 "Veo 3.1 Lite"라는 명칭이 혼용될 수 있습니다. 동일한 모델입니다.

---

## 요약

Veo 3.1 업데이트로 Google Flow의 모델 라인업이 세 단계로 완성되었습니다.

| 목적 | 선택 |
|------|------|
| 빠른 테스트 | Fast Mode (Veo 2) |
| SNS용 중간 품질 + 기본 오디오 | **Veo 3.1 Lite** ← 신규 |
| 최고 품질 + 완전한 오디오 | Veo 3.1 고품질 |

Veo 3.1 Lite의 등장으로 "Fast Mode로 테스트 → Veo 3.1로 최종 완성"이라는 2단계 전략이 **3단계 전략**으로 발전했습니다. 이 3단계 접근법을 익히면 크레딧 효율을 크게 높일 수 있습니다.
