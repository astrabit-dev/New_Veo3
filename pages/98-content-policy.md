# 82. Google 콘텐츠 정책 이해

## VEO 3 금지 콘텐츠 목록

Google은 VEO 3를 포함한 모든 AI 생성 도구에 대해 명확한 콘텐츠 정책을 운영합니다. 다음 유형의 콘텐츠를 생성하려는 시도는 차단되거나 계정 정지로 이어질 수 있습니다.

### 절대적 금지 콘텐츠

| 유형 | 설명 |
|------|------|
| 아동 성착취 (CSAM) | 미성년자를 성적으로 표현하는 모든 콘텐츠 |
| 실존 인물 딥페이크 | 동의 없이 실존 인물의 얼굴/목소리 재현 |
| 테러리즘 선전 | 테러 단체 홍보, 폭력 선동 |
| 대량 살상 무기 정보 | 핵, 생물, 화학 무기 제조 관련 정보 |
| 선거 조작 콘텐츠 | 후보자의 발언을 조작하거나 허위 선거 정보 |
| 자해/자살 조장 | 자해 방법을 구체적으로 보여주거나 권장 |

## 정책 위반 시 결과

### 단계별 제재

**1단계: 콘텐츠 차단**
특정 프롬프트 시도가 차단됩니다. "이 내용을 생성할 수 없습니다"라는 메시지와 함께 생성 거부.

**2단계: 경고**
위반적인 시도가 반복되면 계정에 경고가 부여됩니다.

**3단계: 임시 정지**
심각하거나 반복적인 위반 시 VEO 3 또는 Google One AI 접근이 임시 중단됩니다.

**4단계: 영구 정지**
중대한 위반 시 계정이 영구 정지될 수 있습니다.

## 경계선에 있는 콘텐츠 다루는 법

명백한 금지 영역이 아니더라도 주의가 필요한 "회색 지대"가 있습니다:

### 폭력 장면
- 허용: 판타지 전투, 비사실적인 폭력 (영화/게임 스타일)
- 주의 필요: 사실적으로 묘사된 신체 훼손, 고통의 상세한 묘사
- 금지: 실제 폭력 방법을 보여주거나 특정 대상을 향한 폭력

```
// 허용 가능
"A knight and a dragon engage in an epic fantasy battle, cinematic"

// 주의 필요
"A soldier is shot, realistic graphic wound detail"

// 금지
"A person is violently attacked in a realistic manner"
```

### 성인 콘텐츠
- 허용: 로맨틱한 장면 (비성적), 커플의 포옹/키스 (암시적)
- 금지: 모든 노출 또는 성적인 장면

### 정치 콘텐츠
- 허용: 역사적 사건의 재현, 정치 풍자 (명백히 허구로 표시된 경우)
- 금지: 실존 정치인의 발언 조작, 선거 관련 허위정보

## 안전하게 제작하는 가이드라인

### 1. "허구임을 명확히" 원칙
실존 세계와 혼동될 수 있는 영상에는 항상 허구임을 표시하세요:
- 제목에 "AI Generated", "Fiction" 명시
- 설명란에 "이 영상은 AI로 생성된 가상의 장면입니다" 표기

### 2. 공상의 세계 유지
현실 세계 배경보다 **판타지, SF, 가상의 장소**를 배경으로 하면 현실 오해 가능성이 줄어듭니다.

### 3. 가상 캐릭터 사용
실존 인물을 연상시키는 캐릭터 대신 완전히 가상의 캐릭터를 만드세요.
- "한 남자" → "키가 큰 남자, 파란 눈, 가상의 캐릭터"
- 특정 유명인 이름 → 다른 이름의 가상 인물

### 4. 프롬프트 기록 유지
생성한 프롬프트와 영상을 기록해두면, 나중에 정책 위반 여부를 검토할 때 도움이 됩니다.

## VEO 3의 자동 안전 필터

VEO 3는 내부적으로 다음을 자동으로 감지하고 차단합니다:

- 특정 유명인 이름이 포함된 딥페이크 요청
- 아동 관련 부적절한 요청
- 폭력/혐오 키워드 조합
- 금지된 무기/물질 제조 관련 요청

이 필터는 지속적으로 업데이트됩니다. 이전에 통과된 프롬프트가 나중에 차단될 수도 있습니다.
