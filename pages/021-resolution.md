# 해상도와 업스케일링

> **720p에서 1080p로 — 언제 업스케일이 필요한가**

---

## Veo 3의 기본 해상도

Veo 3로 생성되는 영상의 기본 해상도는 **720p (1280×720 픽셀)**입니다.

720p는 HD(고화질)이지만, 현재 대부분의 플랫폼이 1080p 이상을 권장합니다.

**720p로도 충분한 상황:**
- TikTok, Instagram Reels (모바일에서 시청)
- 빠른 테스트 및 프로토타입
- 내부 검토용 영상
- 낮은 해상도로도 충분한 환경 배경 영상

**1080p가 필요한 상황:**
- YouTube 업로드 (채널 신뢰도를 위해)
- 클라이언트 납품 광고 영상
- 대형 화면(TV, 프로젝터)에서 재생
- 고품질 포트폴리오 영상
- 편집 후 압축을 고려한 원본 영상

---

## 1080p 업스케일 방법

### Google Flow 내 업스케일

영상 생성 후 해당 클립에서 "Upscale to 1080p" 옵션을 선택합니다.

**프로세스:**
1. 720p 영상 생성
2. 마음에 드는 영상 선택
3. "Upscale to 1080p" 클릭
4. 추가 크레딧 소비 (약 10-20 크레딧)
5. 1-3분 대기 후 1080p 버전 제공

### 외부 업스케일 도구 활용

크레딧 절약을 위해 외부 업스케일 AI를 사용하는 방법도 있습니다.

**무료/저렴한 업스케일 도구:**
- **Topaz Video AI**: 영상 전문 업스케일 (유료, 높은 품질)
- **DaVinci Resolve**: 기본 업스케일 기능 포함 (무료)
- **CapCut**: 간단한 해상도 향상 기능 (무료)
- **Waifu2x**: 애니메이션 스타일 영상 업스케일 (무료)

---

## 업스케일의 장단점

### 장점
- 더 선명한 디테일
- 플랫폼 권장 해상도 충족
- 편집 시 크롭/줌 여유 확보
- 시청자 경험 향상

### 단점
- 추가 크레딧 소비 (Flow 업스케일 시)
- 추가 처리 시간
- 파일 용량 증가 (~4배)
- 업스케일 과정에서 미묘한 아티팩트 발생 가능

## 화면 비율별 픽셀 수

| 비율 | 720p | 1080p |
|------|------|-------|
| 16:9 (가로) | 1280×720 | 1920×1080 |
| 9:16 (세로) | 720×1280 | 1080×1920 |
| 1:1 (정방형) | 720×720 | 1080×1080 |

---

## 실용적인 해상도 전략

**권장 워크플로:**
1. 테스트 단계: 720p Fast Mode로 생성 (업스케일 안 함)
2. 확정 단계: Veo 3 고품질로 720p 생성
3. 최종 단계: 납품/공개용만 1080p 업스케일

이렇게 하면 불필요한 업스케일 비용 없이 필요한 영상만 고해상도로 만들 수 있습니다.

## 파일 크기 고려사항

| 해상도 | 8초 클립 예상 파일 크기 |
|--------|----------------------|
| 720p | 약 10-30 MB |
| 1080p | 약 40-100 MB |

여러 클립으로 긴 영상을 만들 계획이라면 저장 공간도 미리 확보해두세요.
