# 요소 5: 장르/스타일 (Genre & Style)

> **한 단어로 수십 개의 시각적 결정을 내립니다**

---

## 장르 키워드의 마법

장르와 스타일 키워드는 Veo 3에게 "이 영상의 전체적인 시각적 언어가 무엇인지"를 알려줍니다.

예를 들어 `film noir`라는 키워드 하나가 의미하는 것:
- 흑백 또는 강한 명암 대비
- 어두운 거리, 형광등, 담배 연기
- 비 내리는 밤, 빗물 반사
- 드라마틱한 그림자
- 탐정, 팜므파탈, 범죄 분위기

이 모든 것을 단 두 단어로 전달합니다. 이것이 장르 키워드의 힘입니다.

---

## 주요 장르/스타일 카탈로그

### 영화/시네마 스타일

#### Cinematic (시네마틱)
```
영어: cinematic, Hollywood style, film quality, movie aesthetic
특징: 영화적 구도, 색 보정, 16:9 레터박스
감정: 서사적, 극적, 완성도 높음
```

#### Film Noir (필름 누아르)
```
영어: film noir, noir aesthetic, classic detective noir
특징: 강한 명암, 어두운 도시, 비, 담배 연기
감정: 미스터리, 범죄, 운명
```

#### Documentary (다큐멘터리)
```
영어: documentary style, cinema vérité, handheld natural
특징: 핸드헬드, 자연광, 비개입
감정: 진실성, 현장감
```

---

### 애니메이션 스타일

#### Pixar / Disney 3D
```
영어: Pixar style, Disney 3D, CGI animation, animated feature film
특징: 밝은 색감, 과장된 표정, 부드러운 라이팅
감정: 따뜻함, 가족친화적, 유머
```

#### Anime (일본 애니메이션)
```
영어: anime style, Japanese animation, manga aesthetic
특징: 큰 눈, 세밀한 배경, 특유의 색감
감정: 다양 (장르에 따라)
```

#### Studio Ghibli
```
영어: Studio Ghibli inspired, Miyazaki style, hand-painted animation
특징: 자연 묘사, 따뜻한 색감, 서정적 움직임
감정: 향수, 자연 경외, 유년기
```

#### Cartoon (만화)
```
영어: cartoon style, 2D animated, Western cartoon, Saturday morning cartoon
특징: 굵은 선, 단순한 색채, 과장된 움직임
감정: 유머, 어린이, 에너지
```

---

### 공상과학 스타일

#### Cyberpunk (사이버펑크)
```
영어: cyberpunk, Blade Runner aesthetic, dystopian future
특징: 네온, 비, 어두운 미래 도시
감정: 위험, 에너지, 인간성 vs 기계
```

#### Space Opera (우주 오페라)
```
영어: space opera, Star Wars style, epic sci-fi
특징: 광활한 우주, 우주선, 행성
감정: 모험, 서사, 운명
```

---

### 판타지 스타일

#### High Fantasy (하이 판타지)
```
영어: high fantasy, epic fantasy, Lord of the Rings style
특징: 마법, 용, 왕국, 고대 세계
감정: 서사, 영웅, 선악 대결
```

#### Dark Fantasy (다크 판타지)
```
영어: dark fantasy, gothic fantasy, Game of Thrones aesthetic
특징: 어둡고 잔혹한 세계, 도덕적 모호함
감정: 긴장, 위험, 복잡함
```

---

### 공포/스릴러 스타일

#### Horror (공포)
```
영어: horror film, psychological horror, slasher aesthetic
특징: 어두움, 점프스케어 예감, 불안감
감정: 공포, 긴장, 불안
```

#### Thriller (스릴러)
```
영어: thriller, suspense, tension-filled, taut atmosphere
특징: 긴장감, 반전, 빠른 전개
감정: 긴장, 흥분, 불확실성
```

---

### 특수 스타일

#### ASMR
```
영어: ASMR style, satisfying, sensory, ultra-close
특징: 익스트림 클로즈업, 세밀한 질감, 부드러운 소리
감정: 이완, 만족감
```

#### Vintage / Retro
```
영어: vintage, retro, 1970s look, nostalgic, aged film
특징: 필름 그레인, 색 바램, 옛날 색감
감정: 향수, 따뜻함, 과거
```

#### Music Video
```
영어: music video aesthetic, MTV style, artistic visual
특징: 빠른 컷, 시각 효과, 예술적 구성
감정: 에너지, 창의성
```

---

## 장르 혼합하기

두 가지 장르 키워드를 조합하면 독특한 스타일이 탄생합니다.

```
Pixar style + horror = 귀여운 픽사 캐릭터들의 공포 이야기
cyberpunk + western = 미래 시대의 총잡이 이야기
anime + documentary = 애니메이션으로 표현된 실사 다큐
```

이런 장르 혼합은 독창적인 콘텐츠를 만드는 좋은 방법입니다.

## 참조 작품 이름 사용

직접 영화나 작품 이름을 참조할 수도 있습니다.

```
"inspired by Blade Runner" (블레이드 러너 분위기)
"Mad Max Fury Road color palette" (매드맥스 색감)
"Werner Herzog documentary style" (베르너 헤어조크 다큐 스타일)
```

단, 특정 IP나 브랜드의 정확한 복제는 저작권 문제가 생길 수 있으므로 "inspired by" 또는 "in the style of"로 표현하세요.
