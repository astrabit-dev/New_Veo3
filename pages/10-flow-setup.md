# Google Flow 30일 무료 시작 단계별 가이드

> **10분이면 준비 완료. 지금 바로 시작하세요.**

---

## 시작 전 필요한 것들

- Google 계정 (Gmail 주소)
- 크롬 브라우저 (권장)
- 신용카드 또는 직불카드 (30일 무료지만 등록 필요)
- 10분의 시간

---

## Step 1: Google Flow 웹사이트 방문

크롬 브라우저를 열고 주소창에 다음을 입력합니다.

```
https://labs.google/flow
```

또는 Google에서 "Google Flow AI video" 검색 후 공식 사이트로 이동합니다.

**화면에 보이는 것:**
- "Google Flow" 로고
- AI 영상 생성 도구 소개 문구
- "Get started" 또는 "Try for free" 버튼

> **주의:** 국가에 따라 접근이 제한될 수 있습니다. 한국에서 접근이 안 된다면 공식 Google 채널에서 서비스 확장 발표를 기다리거나, Google의 공식 가이드를 참고하세요.

---

## Step 2: Google 계정으로 로그인

**2-1.** "Sign in" 또는 "Get started" 버튼 클릭

**2-2.** Google 계정 로그인 화면이 나타남
- 이미 크롬에 Google 계정이 로그인되어 있다면 자동으로 선택됩니다
- 아니라면 Gmail 주소와 비밀번호 입력

**2-3.** Google Flow에 접근 권한 허용

> **팁:** Veo 3 영상은 Google 계정에 연결되어 저장됩니다. 개인 계정보다 크리에이터 전용 계정을 새로 만드는 것을 추천합니다.

---

## Step 3: Google One AI Premium 무료 체험 등록

Google Flow는 Google One AI Premium 구독을 통해 이용합니다.

**3-1.** "Start free trial" 또는 "무료 체험 시작" 버튼 클릭

**3-2.** Google One 구독 페이지로 이동
- 요금제 확인: **$19.99/월** (30일 무료 체험)
- 포함 내용: Google Flow, Gemini Advanced, 2TB 구글 드라이브 등

**3-3.** 결제 정보 입력
- 신용카드 또는 직불카드 번호 입력
- 청구 주소 입력

**3-4.** "Start free trial" 확인 클릭

> **중요:** 30일 무료 체험이 끝나면 자동으로 월 $19.99가 결제됩니다. 계속 사용하지 않을 계획이라면 **달력에 29일 후 알림을 설정**하고 그 전에 취소하세요.

**취소 방법:**
- Google 계정 설정 → 구독 → Google One → 취소
- 언제든지 취소 가능하며 취소 즉시 서비스 중단 (또는 월말까지 유지)

---

## Step 4: Google Flow 대시보드 진입

등록이 완료되면 Google Flow 대시보드가 열립니다.

**처음 보이는 화면 구성:**

```
┌─────────────────────────────────────────┐
│  Google Flow                    [계정]  │
├───────────┬─────────────────────────────┤
│           │                             │
│  사이드바  │     메인 작업 영역           │
│           │                             │
│ [Projects]│  "What will you make?"      │
│ [Flow TV] │                             │
│ [Library] │  [텍스트 입력창]             │
│           │                             │
│           │  [모델 선택] [화면비율]       │
│           │                             │
│           │  [Generate 버튼]             │
│           │                             │
└───────────┴─────────────────────────────┘
```

**각 영역 간단 설명:**
- **Projects**: 작업 중인 프로젝트 목록
- **Flow TV**: 다른 크리에이터들의 영상과 프롬프트 구경
- **Library**: 생성한 영상 보관함
- **메인 영역**: 영상을 만드는 곳

---

## Step 5: 크레딧 확인

대시보드 어딘가에 현재 보유 크레딧이 표시됩니다.

**무료 체험 시작 시:** 1,000 크레딧

이 크레딧으로:
- Veo 3 고품질 (~100 크레딧): 약 10개 클립
- Fast Mode (~10-20 크레딧): 약 50-100개 클립
- 혼합 사용: 약 30-50개 클립 (권장)

---

## Step 6: 첫 번째 프로젝트 만들기

**6-1.** "New Project" 또는 "+" 버튼 클릭

**6-2.** 프로젝트 이름 입력 (예: "My First Videos")

**6-3.** 프로젝트가 생성되면 빈 장면 화면이 나타남

이제 첫 번째 영상을 만들 준비가 된 것입니다!

---

## 자주 겪는 문제들

### "지역에서 서비스 불가" 메시지
한국 등 일부 국가에서는 서비스가 제한될 수 있습니다. Google의 서비스 확장 발표를 기다리거나, 접근 가능한 경로(예: Gemini)를 활용하세요.

### 결제가 안 될 때
- 해외 결제가 가능한 카드인지 확인 (대부분의 Visa, Mastercard 가능)
- 카드사에서 해외 결제를 막은 경우 일시적으로 허용 후 시도

### 로그인이 반복될 때
- 크롬 쿠키 삭제 후 재시도
- 시크릿 모드로 접속해보기

---

## 무료 체험 기간 최대한 활용하는 법

30일, 1,000 크레딧으로 최대한 많이 배우려면:

1. **첫 2주**: Fast Mode로 최대한 많이 실험 (크레딧 절약)
2. **3주차**: 마음에 드는 프롬프트를 Veo 3로 돌려서 품질 비교
3. **마지막 주**: 최고의 영상들을 Veo 3 고품질로 제작
4. **30일 전**: 계속 사용할지 결정
