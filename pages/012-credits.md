# 크레딧 시스템 완전 이해

> **1,000 크레딧을 가장 현명하게 사용하는 방법**

---

## 크레딧이란 무엇인가

Google Flow에서 영상을 생성할 때마다 **크레딧**이 소비됩니다. 월 구독료($19.99)를 내면 매월 **1,000 크레딧**이 지급됩니다.

크레딧은 은행 통장의 잔고처럼 생각하면 됩니다. 영상을 생성할 때마다 잔고가 줄어들고, 다음 달 초에 1,000으로 리셋됩니다. 남은 크레딧은 다음 달로 이월되지 않습니다.

---

## 모드별 크레딧 소비량

| 모드 | 크레딧 소비 | 생성 시간 | 오디오 | 화질 |
|------|-----------|----------|--------|------|
| **Veo 3.1 (고품질)** | ~80-100 크레딧 | 2-5분 | ✅ 완전 포함 | 최상 |
| **Veo 3.1 Lite** | ~30-50 크레딧 | 1-2분 | ✅ 기본 포함 | 매우 좋음 |
| **Fast Mode (Veo 2)** | ~10-20 크레딧 | 30초-1분 | ❌ 없음 | 좋음 |
| **1080p 업스케일** | ~10-20 크레딧 추가 | 1-3분 | - | - |

> **참고:** 정확한 크레딧 소비량은 영상 길이, 해상도, 복잡도에 따라 달라질 수 있습니다. 위의 수치는 평균적인 참고값입니다. Veo 3.1 Lite는 2026년 3월 신규 추가된 중간 등급 모델입니다.

## 1,000 크레딧으로 할 수 있는 것들

### 시나리오 A: Veo 3.1 고품질만 사용할 경우
```
1,000 ÷ 90 = 약 11개 클립
```
- 11개 × 8초 = 최대 88초 분량
- 충분히 고품질이지만 실험 여지가 적음

### 시나리오 B: Veo 3.1 Lite만 사용할 경우
```
1,000 ÷ 40 = 약 25개 클립
```
- 25개 × 8초 = 약 3분 20초 분량
- 기본 오디오 포함, SNS 게시 수준

### 시나리오 C: Fast Mode만 사용할 경우
```
1,000 ÷ 15 = 약 66개 클립 (평균 15크레딧 가정)
```
- 66개 × 8초 = 약 8분 분량
- 실험에 충분하지만 오디오 없고 품질 제한

### 시나리오 D: 3단계 혼합 전략 (권장)
```
Fast Mode로 테스트:   300 크레딧 → 약 20개 실험
Veo 3.1 Lite로 검증: 400 크레딧 → 약 10개 중간 품질
Veo 3.1로 최종 제작: 300 크레딧 → 약 3개 고품질 클립
```
- 테스트 → 검증 → 완성의 3단계 워크플로우
- 이 책에서 추천하는 전략

## 크레딧 소비 추적하기

Google Flow 화면 상단에 항상 현재 크레딧이 표시됩니다. 영상 생성 시 "이 작업에 약 X 크레딧이 소비됩니다"라는 안내가 나타나기도 합니다.

**매일 확인 습관 만들기:**
- 하루를 시작할 때 크레딧 잔고 확인
- 하루 예산 설정 (예: 하루에 최대 100 크레딧 사용)
- 크레딧이 100 이하로 떨어지면 아껴서 사용

## 크레딧 아끼는 기본 전략

### 전략 1: 반드시 Fast Mode로 먼저 테스트
새로운 프롬프트를 처음 시도할 때는 항상 Fast Mode로 먼저 테스트하세요. 결과가 마음에 들면 그때 Veo 3로 고품질 버전을 만드세요.

```
새 프롬프트 → Fast Mode (~15크레딧) → 마음에 들면 → Veo 3 (~100크레딧)
```

이 전략만 지켜도 크레딧 사용을 상당히 절약할 수 있습니다.

### 전략 2: 프롬프트를 충분히 다듬은 후 생성
프롬프트를 8요소 공식에 따라 충분히 작성하고 검토한 뒤 Generate 버튼을 누르세요. 불완전한 프롬프트로 여러 번 Regenerate하는 것보다 처음부터 좋은 프롬프트를 쓰는 것이 훨씬 경제적입니다.

### 전략 3: Regenerate는 2-3번만
같은 프롬프트로 Regenerate를 반복해도 결과가 크게 달라지지 않는 경우가 많습니다. 2-3번 시도해도 마음에 안 들면 프롬프트 자체를 수정하세요.

### 전략 4: 1080p 업스케일은 꼭 필요할 때만
모든 영상을 1080p로 업스케일할 필요는 없습니다. 최종 유튜브 업로드나 광고 납품용 영상만 업스케일하세요.

### 전략 5: Flow TV에서 먼저 검증
새로운 스타일이나 장르를 시도하기 전에 Flow TV에서 비슷한 영상을 찾아보고, 그 프롬프트를 참고하세요. 처음부터 실험하는 것보다 이미 검증된 방식을 활용하면 크레딧 낭비를 줄일 수 있습니다.

## 크레딧이 부족할 때

월 1,000 크레딧을 다 썼다면:

1. **다음 달을 기다리기**: 월 구독이 갱신되면 1,000 크레딧 재충전
2. **Gemini로 일시 전환**: 하루 3개 무료 사용
3. **추가 크레딧 구매**: Google Flow에서 별도 크레딧 팩 구매 가능 (가격 정책 확인 필요)

## 크레딧 관리 도구: 나만의 로그 만들기

영상 생성 기록을 직접 추적하면 크레딧 관리에 도움이 됩니다.

**스프레드시트 예시:**

| 날짜 | 프롬프트 요약 | 모드 | 크레딧 소비 | 결과 평가 | 보관? |
|------|-------------|------|-----------|----------|-------|
| 3/1 | 황금빛 초원 말 | Fast | 15 | ★★★☆☆ | ❌ |
| 3/1 | 빅풋 숲 브이로그 | Fast | 15 | ★★★★☆ | ✅ |
| 3/2 | 빅풋 브이로그 (최종) | Veo 3 | 100 | ★★★★★ | ✅ |

이런 로그를 유지하면 어떤 프롬프트가 좋은 결과를 내는지 파악하고, 나중에 비슷한 영상을 만들 때 참고할 수 있습니다.

## 크레딧 시스템 요약

- 월 $19.99 = 1,000 크레딧
- Veo 3.1 고품질: ~80-100 크레딧/클립
- Veo 3.1 Lite: ~30-50 크레딧/클립 ← 신규
- Fast Mode (Veo 2): ~10-20 크레딧/클립
- **황금 전략: Fast Mode 테스트 → Veo 3.1 Lite 검증 → Veo 3.1 완성**
- 크레딧은 이월되지 않음 (매월 초 리셋)
- 부족하면 Gemini 무료로 보완

크레딧을 현명하게 관리하는 것이 Veo 3를 오래, 효율적으로 사용하는 핵심입니다.
