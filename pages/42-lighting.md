# 요소 4: 조명 (Lighting)

> **빛은 영상의 감정을 조각합니다**

---

## 조명이 영상에서 하는 일

조명은 단순히 "밝게 하는 것"이 아닙니다. 조명은 감정을 만들고, 시선을 유도하고, 시간과 장소를 암시합니다.

영화 감독들은 조명을 "감정의 붓"이라고 부릅니다. 프롬프트에 올바른 조명 키워드를 넣으면 AI가 그 감정에 맞는 빛을 배치합니다.

---

## 조명 유형 완전 가이드

### 자연광 계열

#### 황금 아워 (Golden Hour)
```
키워드: golden hour, golden light, warm amber sunlight,
        setting sun, sunrise glow, magic hour

특징: 낮은 각도의 따뜻한 황금빛
감정: 낭만, 희망, 향수, 아름다움
시간대: 일출 후 / 일몰 전 1시간
```

**예시:**
```
golden hour backlight, warm amber glow, long soft shadows,
lens flare from the setting sun
```

#### 블루 아워 (Blue Hour)
```
키워드: blue hour, twilight light, dusk sky,
        deep blue ambient light, evening glow

특징: 일몰 직후의 깊은 파란빛
감정: 고요함, 신비, 쓸쓸함, 전환
```

#### 한낮 (Harsh Midday)
```
키워드: harsh midday sun, high noon lighting,
        strong overhead light, bleached sunlight

특징: 강하고 직접적인 빛, 강한 그림자
감정: 강렬함, 긴장감, 외로움
```

#### 흐린 날 (Overcast)
```
키워드: overcast sky, diffused light, cloudy day lighting,
        soft even lighting

특징: 부드럽고 균일한 빛, 그림자 거의 없음
감정: 평온함, 우울함, 차분함
```

#### 달빛 (Moonlight)
```
키워드: moonlight, silver moonlight, cold moonlight,
        lunar lighting, moonlit

특징: 차갑고 은빛 빛
감정: 신비, 낭만, 고독, 마법
```

---

### 인공광 계열

#### 촛불 / 화톳불
```
키워드: candlelight, firelight, flickering warm light,
        torch light, campfire glow

특징: 따뜻하지만 불규칙하게 흔들리는 빛
감정: 친밀함, 원시적 에너지, 위험, 따뜻함
```

**예시:**
```
candlelight casting dancing shadows on stone walls,
warm amber and orange tones, flickering light
```

#### 형광등 (Fluorescent)
```
키워드: fluorescent lighting, cool office lighting,
        institutional lighting, sterile white light

특징: 차갑고 균일한 인공 빛
감정: 불편함, 냉정함, 현실적
주요 사용: 사무실, 병원, 편의점, 취조실
```

#### 네온 (Neon)
```
키워드: neon lights, neon glow, neon pink and blue,
        cyberpunk lighting, LED glow

특징: 밝고 강렬한 색상의 인공 빛
감정: 에너지, 위험, 미래적, 현대
```

**예시:**
```
neon lights reflecting on rain-soaked pavement,
deep shadows between pools of colored light
```

#### 스포트라이트 (Spotlight)
```
키워드: single spotlight, dramatic spotlight,
        pool of light in darkness, key light

특징: 하나의 강렬한 빛 아래 강한 그림자
감정: 연극적, 드라마틱, 압박감
주요 사용: 인터로게이션, 무대 공연, 드라마틱 강조
```

## 조명 방향과 효과

| 방향 | 키워드 | 효과 |
|------|--------|------|
| 정면 | front lighting | 자연스럽고 명확한 |
| 측면 | side lighting / Rembrandt | 입체감, 드라마틱 |
| 역광 | backlit, rim lighting | 실루엣, 신비 |
| 위 | overhead / top lighting | 고독, 차가움 |
| 아래 | underlighting | 공포, 기이함 |
| 빛 산란 | diffused light | 부드럽고 균일한 |

## 조명과 감정의 매핑

| 원하는 감정 | 추천 조명 키워드 |
|-----------|---------------|
| 낭만적 | golden hour, candlelight, warm bokeh |
| 공포스러운 | single overhead light, underlighting, deep shadows |
| 신비로운 | moonlight, blue hour, magical glow |
| 영화적 | dramatic backlight, Rembrandt lighting |
| 현실적 | natural daylight, overcast, documentary lighting |
| 고급스러운 | studio lighting, clean white light, product photography |
| 사이버펑크 | neon mixed lights, purple and pink glow |
| 따뜻하고 아늑한 | firelight, golden hour, warm soft window light |

## 고급 조명 기법 키워드

```
god rays / light shafts - 빛기둥 (나무 사이나 창문을 통한 빛)
lens flare - 렌즈 플레어 (빛이 렌즈에 직접 닿는 효과)
bokeh - 보케 (아웃포커스 배경의 빛 흩뿌림)
chiaroscuro - 키아로스쿠로 (강한 명암 대비, 바로크 회화 느낌)
high key lighting - 하이키 (밝고 그림자 적음, 긍정적)
low key lighting - 로우키 (어둡고 그림자 많음, 드라마틱)
```

## 실전 조명 조합 예시

### 탐정 취조실
```
harsh single overhead fluorescent light,
deep shadows under the eyes and chin,
one character in the spotlight, one in shadow
```

### 로맨틱 저녁 식사
```
warm candlelight, soft ambient glow,
shallow depth of field with golden bokeh in background
```

### SF 우주선 내부
```
cool blue ambient lighting, LED strip accents,
harsh shadows, metallic surfaces catching the light
```
