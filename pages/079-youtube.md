# 67. YouTube 장편 영상 구성법

## VEO 3 클립의 한계와 가능성

VEO 3는 현재 한 번에 최대 8초 클립을 생성합니다. YouTube 장편 영상 (5분, 10분, 30분)을 만들려면 여러 클립을 하나로 연결해야 합니다. 이것은 제약처럼 보이지만, 오히려 **영화 편집 기술을 자연스럽게 익히게 되는 장점**이 있습니다.

**수학적 계산:**
- 8초 클립 × 10개 = 80초 영상
- 8초 클립 × 40개 = 5분 20초 영상
- 8초 클립 × 75개 = 10분 영상

각 클립 생성에 약 2-5분이 걸린다면, 10분 영상을 만드는 데 VEO 3 작업만 2.5-6시간입니다.

---

## 여러 클립을 하나의 영상으로 연결하는 방법

### 방법 1: 순차 연결 (Continuity Editing)

같은 장면의 연속성을 유지하면서 클립을 이어붙이는 방법입니다.

예시: 드래곤과 기사가 싸우는 장면
- 클립 1: 기사가 검을 뽑는 장면 (클로즈업)
- 클립 2: 드래곤이 불을 내뿜는 장면 (미디엄샷)
- 클립 3: 기사가 방패로 막는 장면 (클로즈업)
- 클립 4: 전체 전투 장면 (와이드샷)

각 클립이 같은 조명, 색상, 시간대를 공유하면 자연스럽게 연결됩니다.

### 방법 2: B-roll 구조

메인 내레이션 클립 + 보조 시각 클립 구조입니다.

```
[A-roll] 내레이션 또는 메인 행동 클립
[B-roll] 내레이션을 지지하는 시각 클립들

예시: 야생 다큐멘터리
A-roll: 나레이터 보이스오버 설명
B-roll: 동물 행동, 환경, 클로즈업 디테일
```

### 방법 3: 씬-씬 연결 (Scene to Scene)

완전히 다른 장면들을 논리적으로 연결합니다.

```
씬 1: 공항 이별 장면
씬 2: 비행기 창밖 구름
씬 3: 새로운 도시 도착 장면
씬 4: 새로운 생활 시작 장면
```

## 외부 편집 툴 추천

VEO 3에서 생성한 클립들을 연결하려면 외부 편집 툴이 필요합니다:

| 툴 | 플랫폼 | 가격 | 난이도 | 특징 |
|----|--------|------|--------|------|
| **CapCut** | PC/모바일 | 무료 (Pro 유료) | 쉬움 | AI 편집 기능, 자막 자동 생성 |
| **DaVinci Resolve** | PC | 무료 (Studio 유료) | 어려움 | 전문가용, 최고 수준 색보정 |
| **Adobe Premiere Pro** | PC | 월 구독 | 중간 | 업계 표준, 강력한 기능 |
| **iMovie** | Mac/iOS | 무료 | 쉬움 | 입문자용, 직관적 |
| **Kdenlive** | PC | 무료 오픈소스 | 중간 | 비용 없이 전문 기능 |
| **Clideo / Kapwing** | 웹 | 무료 (워터마크) | 매우 쉬움 | 빠른 온라인 편집 |

**초보자 추천**: CapCut → DaVinci Resolve 순서로 배우세요.

## 썸네일 제작

YouTube에서 썸네일은 조회수의 핵심입니다. VEO 3 클립에서 좋은 썸네일 프레임을 만드는 방법:

### 방법 1: 영상에서 스크린샷
VEO 3로 생성한 영상의 가장 임팩트 있는 순간을 정지 화면으로 캡처해 썸네일로 사용합니다.

### 방법 2: 썸네일 전용 프롬프트
영상이 아닌, 썸네일용 정지 이미지를 VEO 3나 Midjourney/Stable Diffusion으로 별도 생성합니다.

썸네일 이미지 프롬프트 예시:
```
A dramatic thumbnail image of a knight facing a massive dragon at sunset, both occupying opposite sides of the frame, epic scale, vibrant warm colors, cinematic composition, high contrast, thumbnail quality
```

### 방법 3: 텍스트 오버레이
썸네일 이미지 위에 CapCut이나 Canva에서 눈에 잘 띄는 텍스트를 추가합니다:
- 큰 폰트, 노란색 + 검은 테두리가 가장 효과적
- 클릭하고 싶은 질문 형식 또는 강한 주장

## 스크립트 작성 팁

YouTube 영상의 스크립트를 먼저 쓰고 VEO 3 클립을 만들면 훨씬 체계적으로 작업할 수 있습니다:

### 10분 영상 스크립트 구조

```
[0:00-0:30] 인트로 (3-4 클립)
  - 훅: 가장 강렬한 장면으로 시작
  - 채널 소개 / 영상 주제 설명

[0:30-2:00] 배경 설명 (12-15 클립)
  - 본 주제의 맥락 설명
  - B-roll로 시각 지원

[2:00-7:00] 메인 콘텐츠 (40-45 클립)
  - 핵심 내용 전달
  - 각 포인트마다 시각 클립

[7:00-9:00] 심화/결론 (15-18 클립)
  - 핵심 요약
  - 심화 정보

[9:00-10:00] 아웃트로 (5-7 클립)
  - 구독/좋아요 요청
  - 다음 영상 예고
```

## 배경음악 추가하기

VEO 3 영상에 배경음악을 추가하려면 저작권 없는 음원을 사용해야 합니다:

**무료 저작권 없는 음원 출처:**
- **YouTube Audio Library** (유튜브 스튜디오 내 무료 음원)
- **Pixabay Music** (pixabay.com/music, 완전 무료)
- **Free Music Archive** (freemusicarchive.org)
- **Artlist** (artlist.io, 유료이지만 퀄리티 높음)
- **Epidemic Sound** (유료, 유튜브 크리에이터 표준)
