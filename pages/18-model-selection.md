# 모델 선택 전략

> **올바른 모델 선택이 크레딧을 아끼고 품질을 높인다**

---

## Google Flow의 세 가지 모드

Google Flow에서 영상을 생성할 때 선택할 수 있는 모드는 크게 세 가지입니다.

1. **Veo 3 (고품질 모드)** — 최상의 품질, 오디오 포함
2. **Fast Mode (Veo 2 기반)** — 빠른 생성, 오디오 없음
3. **Ingredients to Video** — 여러 요소 결합

각각의 특징과 언제 사용해야 하는지 자세히 살펴봅니다.

---

## 모드 1: Veo 3 고품질

### 특징
- **크레딧:** ~100 크레딧/클립
- **생성 시간:** 2-5분
- **오디오:** 포함 (대화, 환경음, 음악)
- **해상도:** 720p (1080p 업스케일 별도)
- **장르 인식:** 탁월
- **물리 법칙:** 최상

### 언제 써야 할까?

✅ **이럴 때 Veo 3를 쓰세요:**
- 유튜브 업로드 또는 SNS 공개할 최종 영상
- 클라이언트 납품용 고품질 광고
- 대사나 환경음이 필요한 영상
- 포트폴리오에 넣을 대표 작품
- 중요한 장면의 최종 버전

❌ **이런 상황에서는 Veo 3가 불필요:**
- 아이디어 테스트용 초안
- 프롬프트가 잘 작동하는지 확인
- 내부 작업용 간단한 배경 클립
- 크레딧이 얼마 안 남았을 때

## 모드 2: Fast Mode (Veo 2)

### 특징
- **크레딧:** ~10-20 크레딧/클립
- **생성 시간:** 30초-1분
- **오디오:** 없음 (묵음)
- **해상도:** 720p
- **품질:** 좋음 (Veo 3보다 낮지만 충분히 쓸 만함)

### 언제 써야 할까?

✅ **이럴 때 Fast Mode를 쓰세요:**
- 새로운 프롬프트 아이디어 빠르게 테스트
- 여러 스타일 중 무엇이 잘 맞는지 실험
- 오디오가 필요 없는 배경 영상
- 크레딧을 아껴야 할 때
- 빠른 납기가 필요한 간단한 작업
- 학습 및 연습 목적

**Fast Mode 크레딧 효율 비교:**
```
Veo 3 100크레딧 = 1개 클립
Fast Mode 15크레딧 = 1개 클립
100크레딧으로: Veo 3 1개 vs Fast Mode 6-7개
```

## 모드 3: Ingredients to Video

### 특징
Ingredients to Video는 여러 요소(이미지, 텍스트, 스타일)를 "재료"로 입력하면 그것들을 조합해 영상을 만드는 방법입니다.

**예시 활용:**
- 제품 사진 + 배경 이미지 + 스타일 설명을 결합해 광고 영상 제작
- 캐릭터 이미지 + 환경 이미지 + 행동 설명을 결합해 스토리 영상 제작

### 언제 써야 할까?

✅ **이럴 때 Ingredients를 쓰세요:**
- 특정 제품/캐릭터의 이미지가 있을 때
- 여러 시각적 요소를 정밀하게 결합하고 싶을 때
- 참조 이미지 기반의 영상 제작
- 캐릭터 일관성이 중요한 시리즈 제작

## 실전 모델 선택 플로차트

```
새 영상 작업 시작
        ↓
   [처음 시도하는 프롬프트?]
   YES → Fast Mode로 테스트
   NO  → 아래로 이동
        ↓
   [Fast Mode 결과 마음에 드는가?]
   NO  → 프롬프트 수정 후 다시 Fast Mode
   YES → 아래로 이동
        ↓
   [최종 공개/납품용 영상인가?]
   NO  → Fast Mode 유지 (충분함)
   YES → 아래로 이동
        ↓
   [오디오(대화/환경음)가 필요한가?]
   NO  → Fast Mode로 최종 생성
   YES → Veo 3 고품질로 생성
```

## 추천 모델 선택 전략: 3단계 접근법

### 1단계: 탐색 (Fast Mode)
새로운 아이디어나 프롬프트는 모두 Fast Mode로 먼저 시도합니다. 빠르고 저렴하게 가능성을 탐색합니다.

### 2단계: 검증 (Fast Mode 반복)
마음에 드는 방향이 나오면 프롬프트를 조금씩 수정하면서 Fast Mode로 계속 테스트합니다.

### 3단계: 완성 (Veo 3)
최종 확정된 프롬프트를 Veo 3로 돌려 최고 품질의 최종본을 만듭니다.

이 3단계 전략을 지키면 1,000 크레딧을 훨씬 효율적으로 사용할 수 있습니다.
