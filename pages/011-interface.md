# Flow 인터페이스 완전 해설

> **버튼 하나하나가 무엇을 하는지, 이제 완벽하게 알게 됩니다**

---

## 전체 화면 구성

Google Flow의 인터페이스는 처음 보면 다소 복잡하게 느껴질 수 있습니다. 하지만 실제로는 논리적으로 구성되어 있습니다. 크게 네 개의 영역으로 나뉩니다.

```
┌─────────────────────────────────────────────────────┐
│  [로고] Google Flow          [크레딧] [계정 아이콘]  │  ← 상단 네비게이션 바
├─────────────────┬───────────────────────────────────┤
│                 │                                   │
│   왼쪽 사이드바  │        메인 작업 영역              │
│                 │                                   │
│  [New Project]  │  ┌─────────────────────────────┐  │
│  ─────────────  │  │      장면 뷰어 / 갤러리        │  │
│  [Projects]     │  │    (생성된 영상 표시)          │  │
│    ├ Project 1  │  └─────────────────────────────┘  │
│    ├ Project 2  │                                   │
│  ─────────────  │  ┌─────────────────────────────┐  │
│  [Flow TV]      │  │         프롬프트 입력창         │  │
│  [Library]      │  │  "Describe your scene..."    │  │
│  [Settings]     │  └─────────────────────────────┘  │
│                 │                                   │
│                 │  [Model] [Ratio] [Duration]       │
│                 │                    [Generate]     │
│                 │                                   │
└─────────────────┴───────────────────────────────────┘
```

---

## 상단 네비게이션 바

### Google Flow 로고
클릭하면 홈 화면(메인 대시보드)으로 돌아갑니다.

### 크레딧 표시
현재 보유 크레딧이 숫자로 표시됩니다. (예: "843 credits")
- 영상을 생성할 때마다 실시간으로 줄어듭니다
- 0이 되면 영상 생성 불가 (구독 갱신 또는 추가 구매 필요)

### 계정 아이콘
클릭하면 계정 설정, 구독 관리, 로그아웃 메뉴가 나타납니다.

---

## 왼쪽 사이드바

### New Project 버튼
새 프로젝트를 만드는 버튼입니다. 프로젝트는 관련된 영상 클립들을 하나의 폴더에 모아두는 개념입니다.

**예시 프로젝트 구조:**
```
Projects
├── 빅풋 브이로그 시리즈
├── 제품 광고 클립들
├── 판타지 단편 영화
└── TikTok 콘텐츠 실험
```

### Projects 섹션
생성한 프로젝트 목록이 여기에 표시됩니다. 클릭하면 해당 프로젝트로 이동합니다.

### Flow TV
다른 크리에이터들이 만든 영상을 구경하는 곳입니다. 가장 중요한 기능 중 하나입니다.

**Flow TV에서 할 수 있는 것:**
- 다른 사람이 만든 영상 감상
- 해당 영상에 사용된 프롬프트 확인 (공개된 경우)
- 카테고리별 탐색 (판타지, 광고, 다큐 등)
- 영감 얻기

### Library
내가 생성한 모든 영상이 보관되는 곳입니다. 프로젝트와 상관없이 전체 생성 기록을 볼 수 있습니다.

---

## 메인 작업 영역

### 장면 뷰어 (Scene Viewer)
현재 프로젝트에서 생성한 영상들이 타임라인 형식으로 표시됩니다.

- 각 영상 클립이 카드 형태로 나열됩니다
- 클릭하면 영상 재생
- 마우스 오버하면 빠른 옵션(다운로드, 삭제, Add to Scene) 표시

### 프롬프트 입력창 (Prompt Input)

**이것이 모든 것의 시작점입니다.**

"Describe your scene..." 또는 "What will you make?"라는 텍스트가 있는 큰 입력창입니다.

여기에 영상 장면 설명을 영어로 입력합니다.

```
A majestic eagle soaring above snow-capped mountains at sunrise,
wide cinematic shot, golden light, dramatic clouds,
slow and powerful wing movements.
[Audio: wind howling, distant eagle cry]
```

**입력 팁:**
- 영어로 작성 (더 좋은 결과)
- 구체적일수록 좋음
- 8요소 공식 활용 (PART 7 참고)

### 모델 선택 (Model Selection)

영상 생성에 사용할 AI 모델을 선택합니다.

| 모델 | 특징 | 크레딧 |
|------|------|--------|
| **Veo 3** | 고품질, 오디오 지원 | ~100 크레딧 |
| **Fast (Veo 2)** | 빠른 생성, 오디오 없음 | ~10-20 크레딧 |
| **Ingredients** | 이미지+텍스트 결합 | 별도 |

### 화면 비율 설정 (Aspect Ratio)

영상의 화면 비율을 선택합니다.

| 비율 | 용도 |
|------|------|
| **16:9** | YouTube, 일반 가로형 영상 |
| **9:16** | TikTok, Instagram Reels, YouTube Shorts (세로형) |
| **1:1** | Instagram 포스트, 정방형 |

### 기간/길이 설정 (Duration)

일부 버전에서는 클립 길이를 조정할 수 있습니다 (보통 5초 또는 8초 옵션).

### Generate 버튼

**파란색 큰 버튼** — 이것을 누르면 영상 생성이 시작됩니다.

- 클릭 후 로딩 인디케이터가 표시됩니다
- Veo 3 고품질: 2-5분 대기
- Fast Mode: 30초-1분 대기
- 완료되면 장면 뷰어에 결과물이 나타납니다

## 영상 생성 후 나타나는 옵션들

영상이 생성되면 각 클립 위에 다음 옵션들이 나타납니다.

### Download (다운로드)
영상을 컴퓨터에 MP4 형식으로 저장합니다. 유튜브 업로드나 편집에 사용할 수 있습니다.

### Add to Scene
이 클립 뒤에 새로운 클립을 추가합니다. 8초 제한을 극복하고 더 긴 영상을 구성하는 방법입니다.

### Regenerate
같은 프롬프트로 다른 버전의 영상을 생성합니다. 마음에 안 드는 결과가 나왔을 때 사용합니다.

### Edit Prompt
프롬프트를 수정하고 다시 생성합니다.

### Upscale to 1080p
720p로 생성된 영상을 1080p로 업스케일합니다. 추가 크레딧이 소모됩니다.

---

## 참조 이미지 업로드 기능

프롬프트 입력창 옆이나 아래에 이미지 업로드 아이콘이 있습니다.

**이 기능으로 할 수 있는 것:**
- 원하는 스타일의 참조 이미지 업로드
- 특정 캐릭터의 외모 참조
- 원하는 장소나 환경 참조
- Frame-to-Video: 정지 이미지를 움직이는 영상으로 변환

---

## 자주 사용하는 단축키 및 팁

1. **Ctrl + Enter** (또는 Cmd + Enter): Generate 버튼과 동일
2. **여러 결과 생성**: 같은 프롬프트로 Generate를 여러 번 눌러 다양한 버전 확인
3. **프롬프트 재사용**: 마음에 드는 영상의 프롬프트를 복사해 수정 후 재사용

---

## 처음 사용자를 위한 권장 순서

1. **Flow TV 먼저 탐색** → 어떤 영상들이 만들어지는지 감 잡기
2. **간단한 프롬프트로 시작** → Fast Mode로 첫 영상 만들기
3. **결과 확인 후 프롬프트 수정** → 더 좋은 결과를 위해 반복
4. **마음에 드는 프롬프트를 Veo 3로** → 최종 고품질 버전 제작

인터페이스에 익숙해지는 데는 2-3번의 영상 생성으로 충분합니다. 두려움 없이 버튼을 눌러보세요!
