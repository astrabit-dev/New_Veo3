# 42. 목소리 톤 설정 방법

VEO 3는 놀랍게도 **캐릭터의 목소리 톤, 억양, 국적, 감정**까지 프롬프트로 제어할 수 있습니다. 같은 대사라도 목소리 설정에 따라 완전히 다른 영상이 만들어집니다.

---

## 목소리 톤의 3가지 구성 요소

### 1. 감정 (Emotion)
캐릭터가 말하는 감정 상태입니다.

| 감정 | 프롬프트 키워드 |
|------|----------------|
| 기쁨 | excited, cheerful, bright |
| 슬픔 | sad, sorrowful, teary |
| 분노 | angry, furious, intense |
| 공포 | terrified, trembling, shaky |
| 자신감 | confident, assertive, commanding |
| 피곤함 | exhausted, weary, tired |
| 긴장 | nervous, anxious, stuttering |
| 차분함 | calm, composed, steady |

### 2. 억양 / 국적 (Accent)
억양을 지정하면 특색 있는 캐릭터가 만들어집니다.

| 억양 | 예시 키워드 |
|------|------------|
| 영국식 | British male/female voice |
| 스코틀랜드식 | Scottish accent, deep and rugged |
| 미국식 | American, casual, upbeat |
| 호주식 | Australian, friendly, relaxed |
| 깊고 울림 있는 | deep, resonant, baritone |
| 높고 맑은 | high-pitched, clear, bright |

### 3. 나이 / 성별 (Age / Gender)
목소리의 나이대와 성별을 지정합니다.

```
예시:
- 젊고 활기찬 20대 남성 목소리
- 거칠고 낮은 60대 노인 목소리
- 부드럽고 따뜻한 중년 여성 목소리
- 어린아이처럼 맑고 호기심 가득한 목소리
```

---

## 목소리 톤 조합 공식

```
[나이] + [성별] + [국적/억양] + [감정] + 목소리로 말한다
```

### 실전 예시 모음

**자신감 있는 지휘관:**
```
자신감 있고 강렬하게 명령하는 40대 미국 남성 목소리로 말한다
```

**긴장한 목격자:**
```
긴장해서 말을 더듬고 빠르게 말하는 20대 젊은 여성 목소리로 말한다
```

**피곤한 탐정:**
```
지치고 건조하며 낮은 50대 영국 남성 목소리로 말한다
```

**유쾌한 빅풋:**
```
깊고 울림 있으며 약간 거친 스코틀랜드 남성 목소리로 말한다
```

**으스스한 악당:**
```
조용하고 차갑게 속삭이는 듯한 낮은 목소리로 천천히 말한다
```

**밝은 브이로거:**
```
에너지 넘치고 친근한 젊은 미국 여성 목소리로 빠르게 말한다
```

## 상황별 목소리 톤 추천

### 브이로그 / 소셜 콘텐츠
- 친근하고 캐주얼한 톤
- 빠르고 에너지 넘치는 전달
- 시청자에게 직접 말하는 느낌

```
에너지 넘치고 친근한 20대 한국계 미국인 남성 목소리로 웃으며 말한다
```

### 드라마 / 단편 영화
- 감정이 실린 자연스러운 연기
- 장면의 긴장감에 맞는 강도

```
두려움과 결의가 뒤섞인 복잡한 감정으로 떨리는 목소리로 말한다
```

### 광고 / 나레이션
- 신뢰감 있고 전문적인 톤
- 명확하고 또렷한 발음

```
따뜻하고 신뢰감 있는 중년 남성 나레이터 목소리로 또렷하게 말한다
```

### 코미디 / 유머
- 과장된 감정
- 예상치 못한 톤 변화

```
매우 진지하게 말하는 척하지만 끝에서 웃음을 참지 못하는 목소리로 말한다
```

## 주의사항

- 너무 많은 수식어를 붙이면 오히려 어색해질 수 있습니다. **3~4가지 핵심 키워드**만 사용하세요.
- 말이 어색하게 생성된다면 대사를 짧게 줄이거나 목소리 톤을 바꿔보세요.
- VEO 3는 영어 대사에서 더 정확한 결과를 보여주는 경향이 있습니다. 한국어 대사는 실험적으로 사용해보세요.
