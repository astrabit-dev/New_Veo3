# 요소 2: 행동 (Action)

> **무엇이 일어나는가 — 영상을 살아있게 만드는 동사의 힘**

---

## 행동 섹션이란

행동(Action)은 캐릭터나 피사체가 무엇을 하는지를 묘사합니다. 영상 프롬프트에서 행동은 정지 이미지를 살아있는 영상으로 만드는 핵심입니다.

**중요한 원칙:** 행동은 동사를 중심으로 작성합니다. 강하고 구체적인 동사가 약하고 모호한 동사보다 훨씬 좋은 결과를 냅니다.

---

## 동사 강도별 분류

### 약한 동사 (피해야 할 것)
```
is (있다)
does (한다)
goes (간다)
moves (움직인다)
looks (본다)
```

### 강한 동사 (사용을 권장)
```
strides confidently (자신 있게 성큼성큼 걷는다)
lurches forward (앞으로 휘청거린다)
gazes into the distance (먼 곳을 응시한다)
crouches silently (조용히 웅크린다)
erupts from the shadows (그림자에서 폭발적으로 나온다)
```

---

## 행동의 3가지 차원

### 차원 1: 물리적 행동
신체가 하는 구체적인 행동

```
물리적 행동 예시:
- sprints at full speed (전속력으로 달린다)
- collapses to one knee (한쪽 무릎으로 무너진다)
- raises both arms triumphantly (두 팔을 의기양양하게 올린다)
- leans against the wall, sliding down slowly (벽에 기대며 천천히 미끄러져 내려온다)
- flips through pages of an old book (오래된 책 페이지를 넘긴다)
```

### 차원 2: 감정적 행동
내면의 감정이 드러나는 행동

```
감정적 행동 예시:
- stares with barely contained rage (거의 억누른 분노로 바라본다)
- smiles to herself, as if remembering something (무언가 기억하는 듯 혼자 미소 짓는다)
- clutches the pendant around her neck, eyes distant (목걸이 펜던트를 꽉 쥐고, 시선은 먼 곳에)
- bites lip nervously, glancing left and right (긴장하여 입술을 깨물며 좌우를 살핀다)
```

### 차원 3: 반응적 행동
외부 자극에 대한 반응

```
반응적 행동 예시:
- freezes when hearing a sound (소리를 듣고 얼어붙는다)
- spins around sharply (갑자기 뒤를 돌아본다)
- ducks behind a pillar (기둥 뒤로 몸을 숨긴다)
- slowly turns to face the camera (천천히 카메라를 향해 돌아선다)
```

---

## 감정적 맥락 추가하기

행동에 감정적 맥락을 추가하면 훨씬 풍부한 영상이 만들어집니다.

**기본 행동:**
```
The detective examines evidence at the crime scene
```

**감정적 맥락 추가:**
```
The detective examines evidence at the crime scene,
hands steady but eyes betraying growing concern,
pausing briefly as something catches her attention,
jaw tightening almost imperceptibly
```

---

## 속도감 표현

행동의 속도는 영상의 에너지를 결정합니다.

**느린 속도:**
```
slowly, deliberately, languidly, in slow motion,
at a glacial pace, each movement measured
```

**빠른 속도:**
```
at breakneck speed, in a blur of motion,
explosively, rapidly, in a flash,
with lightning reflexes
```

**리듬감 있는 속도:**
```
with rhythmic steps, flowing and graceful,
with a steady unhurried cadence
```

## 행동 예시 10개

### 1. 영웅 등장
```
steps through the billowing smoke, sword in hand,
head held high despite the blood on the armor,
strides forward with quiet resolve
```

### 2. 비밀 발견
```
carefully peels back the corner of the painting,
revealing a hidden document behind,
eyes widening with realization
```

### 3. 이별 장면
```
turns for one last look before boarding the train,
lifts a hand in a small uncertain wave,
the smile not quite reaching the eyes
```

### 4. 어린이 놀이
```
chases butterflies across the meadow,
arms outstretched, laughing with abandon,
stumbles and rolls in the grass still laughing
```

### 5. 음식 ASMR
```
slowly slices through the crystalline dessert,
each cut precise and deliberate,
watching as it separates with perfect satisfaction
```

### 6. 빅풋 브이로그
```
holds the camera at arm's length, waving at it,
speaks earnestly while walking through the forest,
stops to pick up and examine a human hiking boot with confusion
```

### 7. 전투 장면
```
blocks the incoming strike with a forearm,
spins into a counterattack,
lands three rapid blows before ducking away
```

### 8. 명상 장면
```
sits cross-legged with eyes closed,
chest rising and falling slowly,
the faintest smile on peaceful lips
```

### 9. 발명 순간
```
stares at the mechanism on the workbench,
suddenly springs forward, adjusting a component,
holds it up as it begins to work
```

### 10. 공포 장면
```
backs slowly away from the door,
never taking eyes off the handle,
breathing quickens as the handle begins to turn
```

## 행동 프롬프트 팁

**팁 1:** 행동을 쓸 때는 항상 "그래서 어떤 감정이 전달되는가"를 함께 생각하세요.

**팁 2:** 여러 개의 작은 행동을 나열하면 더 자연스럽습니다. `"turns, walks to the window, and stares out"` 처럼.

**팁 3:** 행동 속도는 전체 장면의 템포를 결정합니다. 빠른 액션 장면에는 빠른 동사, 감성적인 장면에는 느린 동사를 사용하세요.
