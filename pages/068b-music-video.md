# 뮤직비디오 제작 템플릿

> **Veo 3로 음악에 맞는 영상 클립을 만드는 실전 전략**

---

## 뮤직비디오의 특성

뮤직비디오는 다른 콘텐츠와 다른 접근이 필요합니다.

- **음악이 먼저, 영상이 나중**: 기존 음악 트랙에 맞춰 비주얼을 설계
- **가사·멜로디와의 동기화**: 가사 내용이나 음악 분위기와 영상이 일치해야 함
- **반복·리듬감**: 비트에 맞는 컷 타이밍이 중요
- **아티스트 비주얼**: 가수나 퍼포머의 외형 일관성이 핵심

Veo 3는 8초 클립을 생성하므로, 뮤직비디오는 여러 클립을 CapCut에서 편집하는 방식으로 완성합니다.

---

## 뮤직비디오 제작 기본 전략

### 1단계: 음악 분석 및 씬 설계

먼저 음악을 들으며 영상의 구조를 설계합니다.

```
분석 항목:
- BPM (빠르기): 클립의 컷 타이밍 기준
- 전체 구조: 인트로 / 버스 / 코러스 / 브릿지 / 아웃트로
- 가사 핵심 이미지: 가사에 등장하는 장소, 감정, 오브젝트
- 분위기: 어두운지, 밝은지, 몽환적인지, 역동적인지

BPM 기준 컷 편집 가이드:
- 60~80 BPM (느림): 클립당 4~8초 유지
- 90~110 BPM (중간): 클립당 2~4초
- 120BPM 이상 (빠름): 클립당 1~2초
```

### 2단계: 씬별 프롬프트 작성

각 씬의 분위기에 맞게 별도 프롬프트를 작성합니다.

### 3단계: Veo 3에서 클립 생성

뮤직비디오 클립은 `[Audio: ...]` 블록을 **최소화하거나 생략**합니다. 실제 음악을 CapCut에서 깔 것이기 때문입니다.

```
뮤직비디오용 프롬프트 구성:
- 오디오 묘사 생략 또는 "no dialogue, ambient sound only"
- 비주얼·카메라·조명·분위기에 집중
```

## 뮤직비디오 편집 가이드 (CapCut)

### 비트 동기화

```
1. 음악 파일을 CapCut 타임라인에 먼저 배치
2. Beat Sync 기능 활성화 (타임라인 → 자동 비트 감지)
3. 비트 마커에 클립 컷 포인트를 맞춤
4. 코러스 부분: 더 짧은 컷, 빠른 전환
5. 버스 부분: 더 긴 컷, 부드러운 전환
```

### 전환 효과 선택

| 장르 | 추천 전환 |
|------|---------|
| K-팝 | Zoom Punch, Flash, Glitch |
| 발라드 | Cross Dissolve, Fade |
| 힙합 | Jump Cut, Whip Pan |
| 인디 | Film Burn, Light Leak |
| EDM | Strobe, Color Flash |

### 색 보정

각 장르별 CapCut 색 보정 방향:

```
K-팝 퍼포먼스: 채도 +15, 대비 +20, 쿨한 색온도
감성 발라드: 채도 -10, 따뜻한 색온도, 밝기 +5
힙합: 대비 +25, 채도 유지, 그림자 진하게
인디: 채도 -20~-30, 필름 그레인 필터, 페이드 블랙 포인트
EDM: 채도 최대, Vivid 필터, 네온 강조
```

## 프로 팁

**팁 1 — 아티스트 비주얼 고정**
뮤직비디오에서 아티스트 외모 일관성은 필수입니다. 레퍼런스 이미지를 Google Flow에 업로드하고 모든 씬에 동일한 캐릭터 묘사 블록을 사용하세요. (도구 스택 조합 가이드 참조)

**팁 2 — 퍼포먼스 씬과 내러티브 씬의 조합**
훌륭한 뮤직비디오는 두 가지 유형의 씬을 번갈아 사용합니다:
- **퍼포먼스 씬**: 가수가 카메라를 향해 노래·춤
- **내러티브 씬**: 가사 이야기를 시각적으로 표현

**팁 3 — 클립 다양성 확보**
같은 장면을 여러 각도로 생성하면 편집 유연성이 높아집니다.
- 와이드샷 1개 + 미디엄샷 1개 + 클로즈업 1개 = 3개 클립
- 같은 장면의 세 각도를 번갈아 편집하면 전문적인 결과물
