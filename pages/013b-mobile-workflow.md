# 모바일 워크플로우

> **스마트폰 하나로 — Google Flow 모바일 + CapCut 완전 가이드**

---

## 모바일 워크플로우가 필요한 이유

이 책의 대부분은 PC 브라우저 기준으로 설명합니다. 하지만 많은 분들이 이동 중, 카페에서, 혹은 아예 PC 없이 스마트폰만으로 콘텐츠를 만들고 싶어합니다.

**모바일 워크플로우의 장점:**
- 언제 어디서나 영상 생성 가능
- TikTok, Instagram Reels 업로드와 연결이 빠름
- CapCut 모바일이 소셜 콘텐츠에 최적화됨

**모바일의 한계:**
- 긴 프롬프트 입력이 불편함
- 대용량 클립 다운로드 시 저장 공간 주의
- 정밀한 편집 작업은 PC 버전이 유리

---

## Google Flow 모바일 접속

Google Flow는 전용 앱이 없습니다. **모바일 브라우저**로 접속합니다.

### 접속 방법

```
1. Chrome (또는 Safari) 앱 실행
2. flow.google.com 입력
3. Google 계정 로그인
4. 바탕화면에 추가 (홈 화면 단축키 설정) 권장
```

### 바탕화면 단축키 설정 방법

**Android (Chrome):**
```
1. flow.google.com 접속 후
2. Chrome 우측 상단 ⋮ 메뉴 탭
3. "홈 화면에 추가" 선택
4. 이름 설정 후 추가
```

**iPhone (Safari):**
```
1. flow.google.com 접속 후
2. 하단 공유 버튼(□↑) 탭
3. "홈 화면에 추가" 선택
4. 이름 설정 후 추가
```

## 모바일에서 Google Flow 사용하기

### 프롬프트 입력 팁

모바일에서 긴 프롬프트 입력은 불편합니다. 두 가지 방법으로 해결합니다.

**방법 1: 메모 앱에서 미리 작성 후 붙여넣기**

```
1. 기본 메모 앱(Samsung Notes, Apple Notes, Google Keep) 열기
2. 프롬프트 미리 작성·저장
3. Google Flow 접속 후 복사·붙여넣기
```

**방법 2: 음성 입력 활용**

```
1. 프롬프트 입력창 탭
2. 키보드의 마이크 아이콘 탭
3. 영어로 음성 입력
   (영어 음성 인식 정확도가 높음)
4. 생성된 텍스트 검토 후 수정
```

**방법 3: ChatGPT 모바일에서 프롬프트 생성**

```
1. ChatGPT 앱에서 한국어로 장면 설명
2. "Veo 3용 영어 프롬프트로 변환해줘" 요청
3. 생성된 프롬프트 복사
4. Google Flow 앱에 붙여넣기
```

---

### 모바일에서 클립 생성

PC 버전과 동일한 방식으로 작동합니다. 다만 몇 가지 주의사항이 있습니다.

```
주의사항:
- 생성 중 화면이 꺼지거나 앱이 전환되어도 생성은 계속됨
  (백그라운드 처리)
- 완료 시 브라우저 탭에 알림이 오면 확인
- Wi-Fi 연결 권장 (LTE/5G로도 가능하나 데이터 소모)
- 배터리 절약 모드 해제 권장
```

---

### 모바일에서 클립 다운로드

```
1. 생성 완료된 클립 탭
2. 우측 상단 ⋮ 또는 다운로드 아이콘 탭
3. "Download" 선택
4. 갤러리(사진 앱)에 자동 저장
```

**저장 공간 주의:** Veo 3 클립(8초, 1080p)은 약 30~80MB입니다. 여러 클립을 다운로드할 경우 사전에 저장 공간을 확인하세요.

## CapCut 모바일 편집 워크플로우

### 설치 및 기본 설정

```
1. App Store / Google Play에서 "CapCut" 다운로드 (무료)
2. 앱 실행 → 계정 로그인 또는 건너뛰기
3. 권한 허용: 사진/동영상 접근 허용 필수
```

### 기본 편집 흐름

```
STEP 1: 새 프로젝트 생성
- "새 프로젝트" 탭
- 갤러리에서 Veo 3 클립 선택 (여러 개 선택 가능)
- 화면 비율 설정 (9:16 세로 / 16:9 가로)

STEP 2: 클립 순서 조정
- 타임라인에서 클립 길게 누르기 → 드래그로 순서 이동
- 클립 끝을 드래그해 길이 조정

STEP 3: 전환 효과
- 클립 사이 | 아이콘 탭
- 원하는 전환 효과 선택
- 지속 시간 조정 (0.3~0.5초 권장)

STEP 4: 배경음악 추가
- 하단 메뉴 "오디오" → "음악" 탭
- CapCut 내장 음악 검색 또는 "내 음악"에서 Suno 파일 사용

STEP 5: 자막 추가
- 하단 메뉴 "텍스트" → "자동 자막"
- 또는 "텍스트 추가"로 수동 입력

STEP 6: 내보내기
- 우측 상단 "내보내기" 탭
- 해상도 1080P, 프레임 30fps 선택
- "내보내기" 탭 후 갤러리 저장
```

## 소셜 미디어 직접 업로드

CapCut 모바일에서는 편집 완료 후 플랫폼에 직접 업로드할 수 있습니다.

```
내보내기 화면에서:
- TikTok으로 공유: TikTok 아이콘 탭 → 자동 로그인 연동
- Instagram으로 공유: Instagram 아이콘 탭
- YouTube에 업로드: YouTube 아이콘 탭
```

**장점:** 갤러리 저장 없이 편집 → 업로드가 한 번에 가능합니다.

## 모바일 전용 추천 워크플로우 (30분 숏폼 제작)

TikTok / YouTube Shorts / Instagram Reels 용 30초~1분 영상을 모바일로만 제작하는 최적 경로입니다.

```
[1단계] 기획 — 5분
- Google Keep / Apple Notes에 3~5개 씬 아이디어 메모
- ChatGPT 모바일 앱으로 프롬프트 초안 생성

[2단계] 영상 생성 — 10분
- Google Flow 모바일 접속
- 준비한 프롬프트 붙여넣기
- Veo 3.1 Lite 선택 (크레딧 절약 + 기본 오디오 포함)
- 3~5개 클립 순차 생성

[3단계] 다운로드 — 2분
- 생성된 클립 갤러리 저장

[4단계] 편집 — 10분
- CapCut 새 프로젝트 (9:16 세로)
- 클립 임포트 → 순서 조정
- 배경음악 추가 (CapCut 내장)
- 자막 자동 생성

[5단계] 업로드 — 3분
- CapCut에서 직접 TikTok / Instagram / YouTube 업로드

총 소요: 약 30분
```

## 모바일에서 자주 발생하는 문제

| 문제 | 원인 | 해결법 |
|------|------|--------|
| 생성 중 페이지 사라짐 | 브라우저가 탭 새로 고침 | 브라우저 탭 전환 최소화, 완료 알림 대기 |
| 클립 다운로드 안 됨 | 저장 공간 부족 | 갤러리에서 불필요한 영상 삭제 후 재시도 |
| CapCut 내보내기 느림 | 모바일 성능 한계 | 배경 앱 종료 후 재시도, 충전 중 내보내기 |
| 화면 비율이 잘림 | 비율 설정 오류 | 프로젝트 생성 시 9:16 설정 확인 |
| 음악이 업로드 후 사라짐 | 저작권 감지 | CapCut 내장 "저작권 안전" 음악 사용 또는 Suno 생성 BGM 사용 |
