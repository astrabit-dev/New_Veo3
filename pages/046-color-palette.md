# 요소 8: 색상 팔레트 (Color Palette)

> **색은 감정의 언어입니다**

---

## 색상 팔레트가 왜 마지막인가

8요소 공식의 마지막 요소인 색상 팔레트는 영상의 "마무리 터치"입니다. 앞의 7가지 요소가 장면의 구조를 만든다면, 색상 팔레트는 그 장면에 감정의 물감을 입힙니다.

---

## 기본 색상 팔레트 유형

### 따뜻한 황금 팔레트 (Warm Golden Palette)
```
키워드: warm golden tones, amber and orange palette,
        golden warmth, sun-kissed colors, earthy warm hues

주요 색상: 황금 노랑, 주황, 붉은 갈색, 따뜻한 크림
감정: 행복, 향수, 따뜻함, 희망, 낭만
적합한 장면: 석양, 가족, 로맨스, 수확
```

**프롬프트 예시:**
```
warm golden and amber tones, sunset color palette,
rich earthy warmth, reminiscent of old photographs
```

### 차가운 파란 팔레트 (Cool Blue Palette)
```
키워드: cool blue palette, icy tones, steel grey and blue,
        cold clinical colors, arctic palette

주요 색상: 아이스 블루, 회색, 청록, 은색
감정: 고독, 냉정함, 침착, 공포, 고립
적합한 장면: 미스터리, 공포, 겨울, 우주
```

### 낮은 채도 뮤트 팔레트 (Muted Desaturated Palette)
```
키워드: muted color palette, desaturated, low saturation,
        washed out colors, soft muted tones, faded palette

주요 색상: 모든 색이 채도를 낮춘 버전
감정: 우수, 예술적, 빈티지, 피로함
적합한 장면: 전쟁 드라마, 역사물, 예술 영상
```

### 고대비 네온 팔레트 (High Contrast Neon Palette)
```
키워드: neon color palette, high contrast neon,
        vibrant neon accents, cyberpunk colors,
        electric pink and blue

주요 색상: 네온 핑크, 전기 파랑, 형광 보라, 진한 배경
감정: 에너지, 위험, 미래적, 클럽
적합한 장면: SF, 사이버펑크, 음악 비디오
```

### 흑백 모노크롬 (Monochrome)
```
키워드: black and white, monochrome, grayscale,
        film noir black and white, high contrast B&W

감정: 클래식, 예술적, 드라마틱, 순수
적합한 장면: 다큐, 필름 누아르, 예술 영화
```

### 파스텔 팔레트 (Pastel Palette)
```
키워드: pastel colors, soft pastels, powder pink and blue,
        dreamy pastels, light airy palette

주요 색상: 베이비 핑크, 라벤더, 민트, 크림
감정: 꿈같음, 부드러움, 순수함, 동화
적합한 장면: 로맨스, 판타지, 어린이 콘텐츠
```

### 보석빛 팔레트 (Jewel Tones)
```
키워드: jewel tones, rich saturated colors,
        deep emerald and sapphire, luxurious colors

주요 색상: 에메랄드 그린, 사파이어 블루, 루비 레드, 금색
감정: 고급스러움, 풍요, 왕실, 마법
적합한 장면: 판타지, 고급 브랜드, 역사물
```

## 색상과 감정의 심리학

| 색상 | 감정/연상 |
|------|---------|
| 빨강 | 열정, 위험, 사랑, 분노 |
| 주황 | 에너지, 따뜻함, 창의성 |
| 노랑 | 행복, 희망, 경고 |
| 초록 | 자연, 생명, 평화, 질투 |
| 파랑 | 고독, 신뢰, 차가움, 슬픔 |
| 보라 | 신비, 왕실, 창의성 |
| 핑크 | 부드러움, 로맨스, 어린시절 |
| 검정 | 권력, 미스터리, 죽음, 세련됨 |
| 흰색 | 순수함, 빈 공간, 차가움 |
| 금색 | 고급, 신성, 성공 |

## 영화별 시그니처 팔레트

유명 영화나 TV 쇼의 색상 팔레트를 참조할 수 있습니다.

```
"Wes Anderson color palette"
→ 파스텔, 대칭적, 동화 같은 포화 색감

"Matrix green tint"
→ 모든 장면에 녹색 기운, 차갑고 디지털한 느낌

"Mad Max warm orange sky"
→ 강렬한 주황 하늘, 먼지 낀 황토색 광야

"La La Land pastel dreamlike"
→ 밝고 따뜻한 파스텔, 낭만적 뮤지컬 색감

"Schindler's List desaturated except red"
→ 흑백 위에 하나의 강렬한 색 강조
```

## 색상 팔레트 프롬프트 작성 팁

**1. 구체적인 색 이름 사용:**
```
일반적: "blue colors"
구체적: "deep navy blue and teal accents, with silver highlights"
```

**2. 감정과 연결:**
```
"a melancholic color palette of grey and muted blue,
with occasional flashes of warm amber like fading memories"
```

**3. 대비를 활용:**
```
"stark contrast between warm orange foreground
and cold blue background shadows"
```

**4. 참조 대상 명시:**
```
"color palette reminiscent of Rembrandt oil paintings —
deep shadows, golden light accents"
```

## 색상 팔레트 없을 때 vs 있을 때

**색상 팔레트 없음:**
```
A woman walks through the snow
→ AI가 임의로 색상 결정, 예측 불가
```

**색상 팔레트 있음:**
```
A woman walks through the snow,
monochromatic palette of white and pale grey,
her red coat the only point of color,
creating a poignant visual contrast
→ 강렬한 시각적 임팩트, 감정 전달 명확
```

색상 팔레트는 8요소 중에서도 시청자에게 가장 즉각적으로 감정을 전달하는 요소입니다.
