## 가장 중요한 차이: 오디오 통합

Veo 2와 Veo 3를 구분하는 가장 근본적인 차이는 **오디오**입니다. 이것 하나만으로도 두 버전은 완전히 다른 도구가 됩니다.

### Veo 2 세계에서의 영상 제작 과정
1. Veo 2로 영상 생성 (묵음)
2. 별도의 음악 플랫폼에서 배경음악 다운로드
3. 음향 효과 별도 구매 또는 제작
4. 영상 편집 소프트웨어에서 음악과 영상 합치기
5. 대사가 필요하면 녹음 또는 AI 보이스오버 따로 추가

**소요 시간:** 단순한 1분짜리 영상도 2-3시간

### Veo 3 세계에서의 영상 제작 과정
1. 오디오 설명을 포함한 프롬프트 작성
2. 영상+오디오가 동시에 생성됨

**소요 시간:** 프롬프트 작성 5분 + 생성 대기 2-5분

## 대화 및 립싱크 기능

Veo 3의 또 다른 혁신은 **립싱크가 맞는 대화 생성**입니다.

Veo 2에서는 영상 속 캐릭터가 말하는 것처럼 보이게 하려면, 영상을 만들고 별도의 AI 더빙 도구(예: HeyGen, D-ID)로 립싱크를 입혀야 했습니다. 두 개의 도구를 사용해야 했고, 결과물의 자연스러움도 떨어졌습니다.

Veo 3는 프롬프트에 대사를 직접 넣으면 캐릭터가 그 대사를 말하는 영상이 생성됩니다. 입술 움직임(립싱크)도 대사에 맞게 자연스럽게 이루어집니다.

**예시 프롬프트:**
```
A Bigfoot creature walks through a snowy forest, holding a camera,
speaking directly into the camera: "I can't believe humans think
I don't exist... I'm literally right here."
[Audio: crunching snow footsteps, howling wind, awe-struck voice]
```

이 프롬프트로 빅풋이 실제로 대사를 말하는 영상이 생성됩니다.

## 영상 품질 비교

같은 프롬프트를 Veo 2와 Veo 3에 각각 입력했을 때의 차이를 살펴보겠습니다.

### 프롬프트:
```
A samurai warrior stands in a bamboo forest at dawn,
mist rising from the ground, cherry blossoms falling.
```

### Veo 2 결과물의 특징:
- 기본적인 장면 구성은 정확
- 뱀부 숲의 질감이 약간 인위적
- 안개 표현이 단순함
- 벚꽃 낙하가 자연스럽지 않음

### Veo 3 결과물의 특징:
- 상세한 대나무 질감과 광원
- 안개가 공기처럼 자연스럽게 흘러다님
- 벚꽃 낙하의 물리적 움직임이 사실적
- 이른 새벽 빛의 미묘한 색감 표현
- (만약 오디오 추가 시) 새 소리와 바람 소리 자동 생성

## 비용 비교

Google Flow의 네 가지 모드는 **크레딧 비용이 크게 다릅니다**.

| 모드 | 크레딧 비용 | 생성 속도 | 품질 |
|------|------------|----------|------|
| Fast Mode (Veo 2 기반) | ~10-20 크레딧/클립 | 빠름 (30초-1분) | 좋음 |
| Veo 3.1 Lite | ~30-50 크레딧/클립 | 중간 (1-2분) | 매우 좋음 |
| Veo 3.1 고품질 | ~80-100 크레딧/클립 | 느림 (2-5분) | 최상 |

월 1,000 크레딧 기준:
- **Fast Mode만 사용:** 50-100개 클립 생성 가능
- **Veo 3.1 Lite만 사용:** 20-33개 클립 생성 가능
- **Veo 3.1만 사용:** 약 10-11개 클립 생성 가능
- **3단계 혼합 전략:** Fast(테스트) → Veo 3.1 Lite(검증) → Veo 3.1(완성) — 권장

## 언제 Veo 2(Fast Mode)를 쓸까?

Veo 2(Fast Mode)가 더 적합한 상황이 있습니다.

1. **아이디어 테스트**: 프롬프트가 원하는 결과를 낼지 빠르게 확인할 때
2. **간단한 배경 영상**: 단순한 풍경이나 배경 클립
3. **크레딧이 부족할 때**: 월말에 크레딧이 거의 없을 때
4. **빠른 납기**: 클라이언트에게 급하게 프리뷰를 보여줄 때
5. **실험적 시도**: 새로운 스타일이나 기법을 시도해볼 때

## 언제 Veo 3.1 Lite를 써야 할까? ← 신규

1. **SNS 콘텐츠**: TikTok, Reels, Shorts처럼 기본 오디오가 있는 영상
2. **테스트 이후 검증**: Fast Mode에서 방향을 잡은 후 오디오 포함 버전 확인
3. **대량 제작**: 크레딧을 아끼면서 기본 오디오가 필요한 여러 편 제작
4. **중간 수준 납품**: 최고 품질까지는 필요 없는 내부 시안이나 저예산 프로젝트

## 언제 Veo 3.1을 써야 할까?

1. **최종 납품 영상**: 유튜브 업로드나 클라이언트 납품용 최종본
2. **캐릭터 대사가 있는 영상**: 직접 말하는 캐릭터, 립싱크가 필요할 때
3. **포트폴리오 작품**: 실력을 보여줄 대표작
4. **고품질이 중요한 광고**: 브랜드 이미지가 걸린 영상
5. **영화적 연출**: 섬세한 카메라 워크와 조명 효과가 필요할 때

## Google Flow에서 모델 선택하는 법

Google Flow 인터페이스에서 영상 생성 옵션을 보면 모델을 선택할 수 있습니다.

- **"Veo 3.1"** → 고품질 모드 (약 80-100크레딧)
- **"Veo 3.1 Lite"** 또는 **"Veo 3.1 Fast"** → 경량 모드 (약 30-50크레딧)
- **"Fast"** 또는 **"Veo 2"** → 기존 Fast Mode (10-20크레딧)

세 모드 모두 같은 프롬프트로 순차적으로 테스트해보는 **3단계 전략**이 가장 효율적입니다.

## 요약: 결론

2026년 4월 기준으로 Google Flow의 모델 라인업은 세 단계로 완성되었습니다.

- **Fast Mode (Veo 2)**: 테스트 도구
- **Veo 3.1 Lite**: SNS 콘텐츠 및 검증 도구 ← 2026년 3월 신규
- **Veo 3.1**: 최종 납품 도구

Veo 3.1 Lite의 등장으로 "묵음 Fast Mode → 바로 Veo 3" 라는 극단적인 2단계 선택에서 벗어나, 보다 유연한 **3단계 워크플로우**가 가능해졌습니다.
