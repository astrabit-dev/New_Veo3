# 부록 G. 빠른 참조 치트 시트

> **한 장으로 보는 Veo 3 핵심 — 인쇄해서 작업 공간에 붙여두세요**

---

## 모델 선택 기준

| 상황 | 선택 | 크레딧 |
|------|------|--------|
| 새 아이디어 테스트 | Fast Mode (Veo 2) | ~15 |
| SNS 콘텐츠 최종 완성 | Veo 3.1 Lite | ~40 |
| 유튜브·클라이언트 납품 | Veo 3.1 고품질 | ~90 |

---

## 프롬프트 8요소 체크리스트

```
□ 캐릭터     — 나이, 성별, 외형, 의상
□ 행동       — 구체적인 동작
□ 카메라     — 샷 타입 + 앵글 + 움직임
□ 조명       — 광원 + 색온도
□ 장르·스타일 — 영화 레퍼런스 또는 미학 키워드
□ 모션       — 움직임 속도와 방식
□ 환경       — 장소, 날씨, 시간대
□ 오디오     — [Audio: 환경음, 음악, 대사]
```

---

## 카메라 핵심 키워드

**샷 타입**
```
extreme close-up  /  close-up  /  medium shot
wide shot  /  full body shot  /  establishing shot
```

**앵글**
```
eye level  /  low angle  /  high angle
bird's eye view  /  worm's eye view  /  Dutch angle
```

**움직임**
```
static  /  slow dolly in  /  dolly out
pan left/right  /  tilt up/down  /  tracking shot
crane up  /  handheld  /  360 orbit
```

## 조명 공식

| 원하는 분위기 | 키워드 |
|------------|--------|
| 따뜻하고 감성적 | `golden hour, warm amber, soft diffused` |
| 차갑고 긴장된 | `cold blue fluorescent, desaturated, harsh shadows` |
| 드라마틱 | `single spotlight, deep contrast, chiaroscuro` |
| 신비로운 | `practical lamp, long shadows, low-key lighting` |
| 자연 다큐 | `overcast sky, natural light, no artificial source` |
| SF·미래 | `neon accents, cool blue-green, bioluminescent` |

## 장르별 스타일 키워드

```
영화 같은 느낌    → cinematic, film grain, anamorphic lens flare
공포·스릴러      → horror aesthetic, ominous atmosphere, tension
K-드라마 감성    → Korean drama aesthetic, longing gaze, melancholic
K-팝 MV        → K-pop music video aesthetic, idol performance
다큐멘터리       → documentary style, BBC Earth quality, observational
광고·제품        → commercial product ad, luxury aesthetic, clean
노이르          → film noir, deep shadows, black and white contrast
SF·판타지       → sci-fi futuristic, epic fantasy, world-building scale
```

---

## 오디오 작성 공식

```
[Audio: [배경음/환경음], [음악], [인물 대사 또는 행동음]]

예시:
[Audio: heavy rain, distant jazz piano, woman says softly "I have to go"]
[Audio: forest birds at dawn, wind through leaves, footsteps on gravel]
[Audio: no dialogue, tense ambient score, heartbeat-like bass pulse]
```

---

## 네거티브 프롬프팅 (자주 쓰는 Avoid)

```
화질·스타일:
[Avoid: cartoonish, anime style, CGI look, oversaturated]

카메라:
[Avoid: camera shake, jumpcuts, fisheye distortion]

영상 요소:
[Avoid: text overlay, subtitles, watermarks, extra people in background]

오디오:
[Avoid: background noise, muffled dialogue, volume spike]
```

---

## 트러블슈팅 빠른 참조

| 증상 | 해결 |
|------|------|
| 장면이 의도와 다름 | 장소·인물·시간 각각 구체화 |
| 카메라 구도 안 맞음 | 샷 타입 + 앵글 + 움직임 3개 명시 |
| 오디오 없음 | Veo 3.1 이상 선택 + `[Audio: ...]` 추가 |
| 캐릭터 불일치 | 동일 묘사 블록 복붙 + 참조 이미지 사용 |
| 불필요한 텍스트 | `[Avoid: text overlay, subtitles]` |
| 생성 거부 | 실제 인물 제거, 표현 순화, 길이 200자 이내로 |

## 크레딧 절약 3원칙

```
1. Fast Mode로 먼저 테스트 → Lite로 오디오 검증 → 고품질로 최종
2. SNS 콘텐츠는 Veo 3.1 Lite로 완결 (고품질과 체감 차이 작음)
3. 성공한 프롬프트는 반드시 저장해 재사용
```

## 모바일 빠른 시작

```

2. ChatGPT 앱에서 프롬프트 생성 → 복사
3. Flow에서 Veo 3.1 Lite 선택 → 생성 → 갤러리 저장
4. CapCut 앱 → 새 프로젝트 (9:16) → 편집 → 업로드
```
