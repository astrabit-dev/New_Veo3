# 프롬프트 황금 원칙 7가지

> **이 7가지만 지켜도 영상 품질이 크게 달라집니다**

---

## 황금 원칙이란

수천 개의 프롬프트를 분석하고 실험하며 발견한 공통 법칙들입니다. 어떤 주제를 다루든, 어떤 장르든 이 7가지 원칙을 지키면 결과물의 품질이 상승합니다.

---

## 원칙 1: 구체적으로 써라 (Be Specific)

가장 기본이면서 가장 중요한 원칙입니다. 모호한 표현을 피하고 구체적인 세부사항을 담으세요.

**모호한 표현 → 구체적인 표현:**
- "큰 나무" → "수백 년 된 고목 참나무, 이끼로 뒤덮인 거대한 뿌리"
- "좋은 조명" → "황금 아워, 따뜻한 주황빛 석양, 측광"
- "슬픈 장면" → "눈물이 볼을 타고 흐르는 클로즈업, 고개를 숙이고 어깨가 떨리는"

**기억할 것:** AI는 "좋다", "나쁘다" 같은 주관적 표현보다 시각적으로 묘사 가능한 구체적인 표현을 더 잘 이해합니다.

---

## 원칙 2: 감독처럼 생각해라 (Think Like a Director)

프롬프트를 쓸 때 자신을 영화 감독이라고 생각하세요.

- 카메라는 어디에 있는가?
- 카메라는 움직이는가, 고정되어 있는가?
- 피사체와 카메라의 거리는?
- 어떤 조명을 사용하는가?

이 질문들에 답하면 자연스럽게 더 영화적인 프롬프트가 됩니다.

**예시:**
```
감독으로 생각한 프롬프트:
"카메라는 낮은 앵글에서 시작해 서서히 올려다보며 피사체의 위엄을 강조한다.
배경은 역광으로 처리해 실루엣 효과를 만든다."

영어로:
"Camera starts from a low angle, slowly tilting up to emphasize the figure's grandeur,
dramatic backlighting creating a powerful silhouette effect."
```

---

## 원칙 3: 감정을 담아라 (Include Emotion)

Veo 3는 감정 형용사를 이해하고 그에 맞는 시각적 요소(색조, 카메라 각도, 페이스)를 자동으로 선택합니다.

**감정 키워드 예시:**
```
melancholic (우울한, 쓸쓸한)
triumphant (승리의, 의기양양한)
tense (긴장된)
serene (평온한, 고요한)
ominous (불길한)
joyful (기쁨에 넘치는)
desperate (절박한)
nostalgic (향수 어린)
mysterious (신비로운)
```

감정 키워드가 없는 프롬프트는 감정 없는 중립적인 영상이 나오기 쉽습니다.

---

## 원칙 4: 하나의 핵심에 집중해라 (One Core Idea)

8초 클립에 너무 많은 것을 담으려 하지 마세요. 하나의 강렬한 장면이 여러 개의 약한 장면보다 훨씬 강합니다.

**나쁜 예 (너무 많은 이벤트):**
```
A warrior fights a dragon, then wins, then finds a treasure,
then meets a princess, and they live happily ever after
```

**좋은 예 (하나의 순간에 집중):**
```
A warrior stands face to face with a massive dragon,
moment of tense standoff before the battle begins,
both figures frozen in anticipation
```

---

## 원칙 5: 장르 키워드를 활용해라 (Use Genre Keywords)

Veo 3는 장르 키워드를 이해하고 그에 맞는 종합적인 시각 스타일을 적용합니다.

**효과적인 장르 키워드:**
```
cinematic (영화적)
film noir (필름 누아르)
Pixar style (픽사 스타일)
anime style (애니메이션)
documentary (다큐멘터리)
horror (공포)
epic fantasy (서사 판타지)
sci-fi (SF)
ASMR (ASMR)
vintage (빈티지)
```

장르 키워드 하나가 수십 개의 세부 설명을 대신할 수 있습니다.

---

## 원칙 6: 움직임을 기술해라 (Describe Motion)

영상은 정지 이미지가 아닙니다. 무엇이 어떻게 움직이는지 기술하면 더 역동적인 영상이 만들어집니다.

**좋은 모션 키워드:**
```
slowly rotating (천천히 회전)
billowing in the wind (바람에 펄럭이는)
swirling dust (소용돌이치는 먼지)
gentle ripples (잔잔한 파문)
cascading water (쏟아지는 물)
crackling fire (타오르는 불꽃)
floating debris (떠다니는 잔해)
```

환경의 모션 뿐만 아니라 캐릭터의 동작도 구체적으로 기술하세요.

---

## 원칙 7: 오디오를 잊지 마라 (Don't Forget Audio)

Veo 3 고품질 모드를 사용할 때는 반드시 오디오 프롬프트를 추가하세요. 오디오가 있는 영상과 없는 영상의 감동 차이는 엄청납니다.

**오디오 추가 방법:**
```
[Audio: 소리 설명]
```

**예시:**
```
[Audio: howling wind, distant thunder,
slow heartbeat increasing in pace]
```

오디오 없이 고품질 모드를 사용하는 것은 크레딧을 낭비하는 것과 같습니다.

---

## 황금 원칙 요약 체크리스트

프롬프트를 쓴 후 이 체크리스트를 확인하세요.

- [ ] 구체적인 세부사항이 포함되어 있는가?
- [ ] 카메라 정보(앵글, 움직임)가 있는가?
- [ ] 감정/분위기 키워드가 포함되어 있는가?
- [ ] 하나의 핵심 장면에 집중되어 있는가?
- [ ] 장르 키워드가 있는가?
- [ ] 움직임 묘사가 있는가?
- [ ] 오디오 설명이 있는가? (Veo 3 모드일 때)

7개 중 5개 이상 체크가 되면 좋은 프롬프트입니다. 모든 7개를 만족하면 탁월한 프롬프트입니다.
