# 색 보정과 조명 제어

> **빛과 색이 감정을 만듭니다**

---

## 조명이 영상에서 하는 역할

좋은 영상 감독은 "조명으로 이야기한다"고 말합니다. 같은 장면이라도 조명에 따라 완전히 다른 감정이 전달됩니다.

- 따뜻한 황금빛 → 행복, 향수, 로맨스
- 차가운 파란빛 → 고독, 슬픔, 미스터리
- 강렬한 네온 → 에너지, 위험, 미래
- 부드러운 자연광 → 평화, 진정성, 일상

Veo 3는 이런 조명 효과를 프롬프트 키워드로 제어할 수 있습니다.

---

## 자연광 조명

### 황금 아워 (Golden Hour)
- **시간대:** 일출 후 1시간, 일몰 전 1시간
- **색감:** 따뜻한 황금빛 주황, 분홍
- **그림자:** 길고 부드러운 그림자
- **감정:** 낭만적, 따뜻함, 서사적

**프롬프트 키워드:**
```
golden hour lighting, warm amber glow, long soft shadows,
sun setting on the horizon, golden light filtering through
```

**예시:**
```
A couple walks hand in hand through wheat fields,
golden hour lighting, warm amber glow surrounding them,
soft lens flare, romantic and nostalgic atmosphere.
```

### 블루 아워 (Blue Hour)
- **시간대:** 일몰 직후, 완전한 어둠 전
- **색감:** 깊은 파란색, 보라색
- **감정:** 고요함, 신비로움, 쓸쓸함

**키워드:**
```
blue hour, twilight, dusk lighting, deep blue sky,
afterglow, transitional light
```

### 한낮 (Harsh Midday Sun)
- **특징:** 강하고 직접적인 빛, 강한 그림자
- **감정:** 강렬함, 긴장감, 사막 같은 외로움

**키워드:**
```
harsh midday sun, strong overhead light, deep shadows,
high contrast, bleached out whites
```

### 흐린 날 (Overcast)
- **특징:** 부드럽고 균일한 빛, 그림자 없음
- **감정:** 평온함, 우울함, 고요함

**키워드:**
```
overcast sky, soft diffused light, no harsh shadows,
even lighting, muted tones
```

## 인공 조명

### 네온 빛 (Neon Lighting)
- **색감:** 밝은 핑크, 파란, 보라, 초록
- **환경:** 도시, 바, 클럽, 미래 세계
- **감정:** 에너지, 위험, 현대성

**키워드:**
```
neon lights, neon glow, neon pink and blue,
cyberpunk lighting, neon reflections on wet streets
```

### 촛불 / 화톳불 (Candlelight / Fire)
- **색감:** 따뜻한 주황빛, 노란빛, 흔들리는 빛
- **감정:** 친밀함, 위험, 원시적

**키워드:**
```
candlelight, firelight, flickering warm light,
dancing shadows, intimate warm glow
```

### 형광등 (Fluorescent Light)
- **색감:** 차갑고 청록빛
- **환경:** 사무실, 병원, 편의점
- **감정:** 불편함, 냉정함, 현실성

**키워드:**
```
fluorescent lighting, cool overhead lighting,
office lighting, hospital lighting, harsh artificial light
```

### 달빛 (Moonlight)
- **색감:** 차가운 파란색/회색
- **특징:** 부드럽지만 신비로운 빛
- **감정:** 신비, 고독, 로맨스

**키워드:**
```
moonlight, silver moonlight, cold blue moonlight,
shadows cast by moonlight, moonlit landscape
```

## 색상 팔레트 제어

조명 외에도 전반적인 색조를 직접 지정할 수 있습니다.

### 따뜻한 팔레트
```
warm color palette, amber and orange tones,
golden and red hues, earthy warm colors
```
감정: 편안함, 향수, 행복

### 차가운 팔레트
```
cool color palette, blue and cyan tones,
icy cold colors, steely grey and blue
```
감정: 차분함, 고독, 침착함

### 낮은 채도 / 뮤트 (Muted)
```
muted color palette, desaturated, low saturation,
faded colors, soft pastel tones
```
감정: 차분함, 예술적, 빈티지

### 고채도 (Vibrant)
```
vibrant colors, high saturation, bold colors,
vivid and punchy palette
```
감정: 활기참, 에너지, 어린이 콘텐츠

### 모노크롬 (흑백)
```
black and white, monochrome, grayscale,
film noir black and white, high contrast black and white
```
감정: 클래식, 예술적, 드라마틱

## 색 보정 스타일

특정 영화나 시대의 색 보정 스타일을 참조할 수도 있습니다.

```
Blade Runner color grading      - 어두운 사이버펑크 톤
Wes Anderson color palette      - 파스텔, 대칭적, 동화 같음
Saving Private Ryan film look   - 탈색, 진흙색, 거친 전쟁
La La Land color grading        - 밝고 따뜻한 뮤지컬 톤
Mad Max Fury Road colors        - 주황 하늘, 강렬한 대비
```

## 조명 방향

빛이 어디서 오는지도 감정에 영향을 미칩니다.

| 조명 방향 | 영어 키워드 | 효과 |
|----------|-----------|------|
| 앞에서 | front lighting | 명확함, 자연스러움 |
| 뒤에서 | backlit, rim lighting | 실루엣, 신비로움 |
| 옆에서 | side lighting | 드라마틱, 입체감 |
| 아래에서 | under lighting | 공포, 기이함 |
| 위에서 | overhead lighting | 고독, 차가움 |
