# 부록 E. 외부 편집 툴 연동 워크플로우

> **Veo 3에서 완성 영상까지 — 단계별 실전 가이드**

---

## 왜 외부 편집 툴이 필요한가

Veo 3는 강력한 영상 생성 도구이지만, 생성된 클립을 그대로 업로드하는 경우는 드뭅니다. 대부분의 완성 영상은 다음 작업을 필요로 합니다:

- 여러 클립 연결 및 순서 조정
- 자막 및 텍스트 삽입 (Veo 3가 약한 부분)
- BGM 및 효과음 추가/조정
- 색 보정 및 필터
- 화면 비율 조정 (TikTok, YouTube, Instagram 등)
- 인트로/아웃트로 삽입

이 부록에서는 **Veo 3 → CapCut** 워크플로우를 중심으로, 상황에 따라 DaVinci Resolve와 Adobe Premiere를 활용하는 방법도 소개합니다.

---

## 편집 툴 선택 가이드

| 도구 | 가격 | 난이도 | 추천 대상 |
|------|------|--------|---------|
| **CapCut** | 무료 (프리미엄 있음) | 쉬움 | 초보자, 모바일/소셜 콘텐츠 |
| **DaVinci Resolve** | 무료 (스튜디오 유료) | 중간 | 색 보정 중요한 작업 |
| **Adobe Premiere** | 월 구독 | 어려움 | 전문 영상 제작자 |
| **iMovie** | 무료 (Mac) | 매우 쉬움 | Mac 사용자, 간단한 편집 |

**초보자에게는 CapCut을 강력히 추천합니다.** 무료이고 직관적이며, 소셜 미디어에 최적화된 기능이 많습니다.

---

## Veo 3 → CapCut 워크플로우

### STEP 1: Veo 3에서 클립 생성 및 다운로드

**클립 생성 전 계획 세우기:**
```
영상 구성 예시 (1분 유튜브 숏츠):
- 클립 1: 오프닝 장면 (8초)
- 클립 2: 주요 장면 A (8초)
- 클립 3: 주요 장면 B (8초)
- 클립 4: 클라이맥스 (8초)
- 클립 5: 엔딩 (8초)
총 5개 클립 × 8초 = 40초 원본
(편집 후 전환 효과 포함 약 45-50초)
```

**다운로드 방법:**
1. Google Flow에서 생성된 클립 우측 상단 `⋮` 메뉴 클릭
2. `Download` 선택
3. 모든 클립을 하나의 폴더에 모아두기 (예: `프로젝트명_클립`)

> **팁:** 클립 파일명을 `clip_01.mp4`, `clip_02.mp4` 형식으로 미리 정리해두면 편집 시 순서 파악이 쉽습니다.

---

### STEP 2: CapCut에서 새 프로젝트 생성

**PC 버전 기준:**

1. CapCut 실행 → `새 프로젝트` 클릭
2. 다운로드한 클립들을 모두 선택하여 임포트
3. 원하는 화면 비율 설정:
   - **TikTok / YouTube Shorts**: 9:16 (세로)
   - **YouTube 일반**: 16:9 (가로)
   - **Instagram 정사각형**: 1:1

---

### STEP 3: 클립 배치 및 순서 조정

1. 임포트된 클립을 타임라인에 순서대로 드래그
2. 각 클립의 시작/끝 트리밍 (불필요한 부분 제거)
3. 클립 간 전환 효과 추가:

**추천 전환 효과:**
```
자연스러운 연결: Cross Dissolve (크로스 디졸브)
역동적인 전환: Zoom In/Out
감성적 장면: Fade to Black
액션 장면: Whip Pan
```

> **팁:** 전환 효과는 0.3~0.5초가 가장 자연스럽습니다. 너무 길면 지루해 보입니다.

---

### STEP 4: 자막 및 텍스트 삽입

Veo 3가 약한 텍스트 삽입을 CapCut에서 처리합니다.

**자동 자막 생성 (CapCut AI 기능):**
1. 타임라인에서 오디오가 있는 클립 선택
2. 상단 메뉴 `자막` → `자동 자막` 클릭
3. 언어 선택 (한국어 또는 영어)
4. 생성된 자막 검토 및 수정

**수동 텍스트 삽입:**
1. `텍스트` → `텍스트 추가`
2. 원하는 위치에 텍스트 박스 배치
3. 폰트, 크기, 색상 조정
4. 텍스트가 표시될 시간 구간 설정

**자막 스타일 추천:**
```
소셜 미디어용: 굵은 폰트, 흰색 텍스트 + 검정 테두리
영화적 느낌: 얇은 폰트, 하단 중앙, 반투명 배경
```

---

### STEP 5: BGM 및 효과음 추가

**BGM 추가:**
1. `오디오` → `음악` 클릭
2. CapCut 내장 음악 라이브러리 검색 (저작권 무료)
3. 또는 직접 MP3 파일 임포트
4. BGM 볼륨을 원본 영상 오디오보다 낮게 조정 (약 20~40%)

**Veo 3 오디오와 BGM 믹싱:**
- Veo 3가 생성한 음향이 있는 클립: BGM 볼륨 더 낮추기 (15~25%)
- Veo 3 음향이 없는 클립: BGM 볼륨 높이기 (40~60%)

**효과음 추가:**
1. `오디오` → `효과음`
2. 장면에 맞는 효과음 선택 (whoosh, hit, ambient 등)
3. 정확한 타이밍에 배치

---

### STEP 6: 색 보정

CapCut의 기본 색 보정으로도 충분히 영상 품질을 높일 수 있습니다.

**기본 색 보정 순서:**
1. 클립 선택 → `조정` 탭
2. **밝기**: 어두운 장면은 +5~15
3. **대비**: +10~20 (선명도 향상)
4. **채도**: +5~15 (색감 풍부하게)
5. **색온도**: 따뜻한 느낌 원하면 +, 시원한 느낌은 -

**필터 활용:**
- 영화적 느낌: `Cinematic` 계열 필터
- 빈티지 느낌: `Film` 계열 필터
- 선명한 느낌: `Vivid` 계열 필터

> **팁:** 필터 강도는 30~50%로 낮추는 것이 자연스럽습니다. 100%는 과도하게 보일 수 있습니다.

---

### STEP 7: 최종 내보내기

**내보내기 설정:**
```
해상도: 1080p (기본값, 충분함)
프레임률: 30fps (일반 콘텐츠) / 60fps (게임, 액션)
품질: 높음
형식: MP4 (범용)
```

**플랫폼별 최적 설정:**
| 플랫폼 | 비율 | 해상도 | 최대 길이 |
|--------|------|--------|---------|
| YouTube Shorts | 9:16 | 1080×1920 | 60초 |
| TikTok | 9:16 | 1080×1920 | 10분 |
| Instagram Reels | 9:16 | 1080×1920 | 90초 |
| YouTube 일반 | 16:9 | 1920×1080 | 무제한 |

## DaVinci Resolve 활용 (색 보정 심화)

CapCut으로 기본 편집을 마친 후, 색 보정에 더 투자하고 싶다면 DaVinci Resolve를 사용하세요.

**DaVinci Resolve가 유리한 경우:**
- 영화적 색감이 중요한 고품질 콘텐츠
- 여러 클립의 색감을 일관되게 맞춰야 할 때
- 광고, 기업 영상 등 전문 납품물

**기본 워크플로우:**
1. CapCut에서 편집 완료 → ProRes 또는 고품질 MP4로 내보내기
2. DaVinci Resolve에서 임포트
3. Color 페이지에서 색 보정
4. Deliver 페이지에서 최종 내보내기

---

## 자주 발생하는 문제와 해결법

| 문제 | 원인 | 해결 |
|------|------|------|
| 클립 간 색감이 다름 | Veo 3 생성 시 편차 | CapCut 색 보정으로 통일 |
| 오디오 볼륨이 들쑥날쑥 | 클립마다 음향 레벨 다름 | 각 클립 볼륨 수동 조정 |
| 자막이 배경과 안 보임 | 폰트 색상 문제 | 텍스트 테두리 또는 배경 추가 |
| 파일 용량이 너무 큼 | 내보내기 설정 문제 | 비트레이트 낮추기 (10~15Mbps) |
| 영상이 흔들려 보임 | 핸드헬드 스타일 클립 | CapCut 손떨림 보정 기능 적용 |
