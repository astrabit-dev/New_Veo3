# 44. 자막 방지 팁

VEO 3로 대화 영상을 만들다 보면 영상 위에 자막이 자동으로 표시되는 경우가 있습니다. 이 현상의 원인과 방지법을 알아봅니다.

---

## 자막이 생기는 주요 원인

### 1. 따옴표 사용
대사에 따옴표를 쓰면 AI가 이를 "화면에 표시할 텍스트"로 인식합니다.

```
❌ 문제 있는 프롬프트:
빅풋이 말한다: "얘들아, 오늘 너무 춥다."

✅ 올바른 프롬프트:
빅풋이 말한다: 얘들아 오늘 너무 춥다.
```

### 2. "자막", "텍스트" 단어 포함
프롬프트에 자막 관련 단어가 있으면 영상에 텍스트가 삽입됩니다.

```
❌ 피해야 할 단어: subtitle, caption, text, 자막, 텍스트
```

### 3. "라고 말했다", "라고 외쳤다" 형식
인용문 형식의 서술체도 자막을 유발할 수 있습니다.

---

## 자막 방지 체크리스트

| 항목 | 올바른 방법 |
|------|------------|
| 따옴표 | 절대 사용하지 않기 |
| 대사 표기 | 일반 텍스트로 자연스럽게 |
| 프롬프트 끝 | "자막 없음" 명시 |
| 텍스트 요청 | 로고, 브랜드명 등 텍스트 삽입 요청 금지 |

---

## "자막 없음" 명시하기

프롬프트 맨 끝에 다음 문구를 추가하세요:

```
자막 없음.
```
또는 영어로:
```
No subtitles. No text overlays.
```

---

## 자막이 생겼을 때 대처법

1. **프롬프트 수정**: 따옴표 제거, "자막 없음" 추가
2. **대사 단축**: 긴 대사는 짧게 줄이기
3. **재생성**: 같은 프롬프트로 다시 생성 (결과가 매번 달라짐)
4. **편집 툴 사용**: 생성 후 영상 편집 툴로 자막 영역 크롭

---

## 올바른 대화 프롬프트 예시

```
키 크고 털이 많은 빅풋이 눈보라 속 숲에서 셀카봉을 들고 걷는다.
셀카 카메라 앵글. 흔들리는 핸드헬드 스타일.
빅풋이 깊고 거친 스코틀랜드 남성 목소리로 말한다:
얘들아 오늘 너무 춥고 다리가 얼어붙을 것 같아 솔직히.
배경음: 강한 바람 소리, 나뭇가지 삐걱거림.
자막 없음.
```
