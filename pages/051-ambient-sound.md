# 43. 배경음·환경음·음악 삽입

VEO 3는 영상에 소리까지 자동으로 생성해줍니다. 배경음을 잘 활용하면 시청자의 몰입감이 완전히 달라집니다.

---

## 오디오의 3가지 종류

### 1. 환경음 (Ambient Sound)
장소와 상황에 맞는 자연스러운 소음입니다.

| 환경 | 환경음 예시 |
|------|------------|
| 숲 / 자연 | 새 지저귐, 바람 소리, 나뭇잎 바스락거림 |
| 도시 | 자동차 소리, 사람들의 웅성거림, 경적 |
| 실내 | 에어컨 소리, 키보드 타이핑, 시계 초침 |
| 해변 | 파도 소리, 갈매기 울음, 바람 소리 |
| 우주선 | 기계 윙윙거림, 경보음, 전자음 |
| 전쟁터 | 폭발음, 총성, 헬리콥터 소리 |

### 2. 특정 사운드 이벤트 (Sound Events)
특정 순간에 발생하는 소리입니다.

```
예시:
- 천둥소리와 번개
- 문이 삐걱거리며 열리는 소리
- 유리 깨지는 소리
- 발소리가 점점 가까워지는 소리
- 동굴 메아리
```

### 3. 배경 음악 (Background Music)
분위기를 완성하는 음악입니다.

| 장르 | 키워드 |
|------|--------|
| 서사적 | epic orchestral music, sweeping strings |
| 긴장감 | tense suspenseful music, low drones |
| 감성적 | soft piano melody, gentle acoustic guitar |
| 신나는 | upbeat electronic music, energetic beats |
| 공포 | eerie horror ambient, creeping strings |
| 자연 다큐 | calm nature documentary score |
| 로파이 | lo-fi hip hop beats, chill background music |
| ASMR | no music, complete silence |

## 프롬프트에 오디오 삽입하는 방법

### 방법 1: 문장 끝에 추가
```
...젊은 여성이 절벽 끝에 서 있다. 로우 앵글 샷.
배경음: 부드러운 바람 소리, 멀리서 들리는 새소리.
배경 음악: 잔잔한 피아노 멜로디.
```

### 방법 2: 환경 설명 안에 통합
```
눈보라가 치는 빽빽한 숲. 바람이 나무 사이를 휘몰아치며
나뭇가지가 삐걱거린다. 쌓이는 눈의 무게에 나무가 신음한다.
```

### 방법 3: 사운드 이벤트로 지정
```
...문이 천천히 열리는 소리와 함께 어두운 복도가 드러난다.
멀리서 발소리가 들린다. 점점 가까워진다.
배경 음악 없음. 완전한 정적.
```

## 장르별 오디오 조합 추천

### 판타지 서사
```
배경음: 바람 소리, 불꽃 타닥거리는 소리
배경 음악: 웅장한 오케스트라, 코러스
```

### 공포 / 스릴러
```
배경음: 삐걱거리는 나무 바닥, 멀리서 들리는 발소리
배경 음악: 으스스한 저음 드론, 간헐적 현악기
```

### 브이로그 / 일상
```
배경음: 카페 소음, 가벼운 잡담 소리
배경 음악: 밝고 캐주얼한 로파이 비트
```

### ASMR / 힐링
```
배경음: 빗소리, 나뭇잎 스치는 소리
배경 음악: 없음 (No music, no dialogue)
```

### 액션 / 전쟁
```
배경음: 폭발음, 헬리콥터, 빗속 달리는 소리
배경 음악: 강렬한 타악기, 전쟁 드럼
```

---

## 핵심 팁

- 배경음은 **최대 2~3가지**만 선택하세요. 너무 많으면 AI가 혼동합니다.
- **음악 없음(No music)**이 효과적일 때도 많습니다. 특히 ASMR, 공포, 드라마 장면에서.
- 오디오 설명은 프롬프트 **끝 부분**에 배치하는 것이 좋습니다.
- 대화가 있는 장면에서 배경 음악을 너무 강하게 설정하면 대사가 묻힐 수 있습니다.
