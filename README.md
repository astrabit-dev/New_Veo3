Google Veo 3는 텍스트 한 줄로 영화 같은 영상을 만들 수 있는 구글의 최신 AI 영상 생성 모델입니다.
이 책은 Veo 3를 처음 접하는 분들이 **설정 → 첫 영상 → 프롬프트 작성 → 실전 제작**까지 막힘 없이 배울 수 있도록 설계되었습니다.

---

## 이 책에서 배울 수 있는 것

- Google Flow 접속 및 무료 체험 시작 방법
- 텍스트로 영화 같은 영상을 만드는 **프롬프트 공식 8요소**
- 대화·오디오·카메라 연출 완전 정복
- TikTok·Instagram·YouTube 플랫폼별 최적화 전략
- JSON 고급 프롬프트로 제품 광고 영상 제작
- AI 영상으로 수익을 창출하는 다양한 방법
- Veo 3.1 / Veo 3.1 Lite 최신 모델 활용법
